# 제2장 공공조달 입찰 실행절차

## 개요

공공조달 입찰 실행절차는 조달계획수립과 시장조사를 거친 후 입찰·낙찰 단계로 진입하는 과정이다. 수요기관의 발주부터 계약체결까지 약 **20개 내외의 세부 절차**로 수행된다.

## 수요기관 발주절차 (9단계)

| 단계 | 구분 | 예상소요기간 |
|:----:|------|:----------:|
| 1 | 발주계획 수립 | 사전준비 |
| 2 | 계약심사·일상감사 의뢰 | D+1 |
| 3 | 계약심사 결과 통보 | D+3 |
| 4 | 일상감사 결과 통보 | D+7 |
| 5 | 계약심사 결과 확정 통보 | D+8 |
| 6 | 사업추진계획 수립 | D+11 |
| 7 | 내부 계약의뢰 | D+13 |
| 8 | 조달청 계약의뢰 | D+18 |
| 9 | 사전규격 등록 | D+23 |

사전규격공개 기간: 일반 **5일** / 긴급 **3일** / SW사업 **10일**

## 입찰공고 기간

| 적용 법령 | 기간 |
|-----------|------|
| 국가계약법 (일반) | 입찰마감일 **7일 전** |
| 국가계약법 (협상계약) | 입찰마감일 **40일 전** |
| 지방계약법 (1억 미만) | **10일** |
| 지방계약법 (1억~10억) | **20일** |
| 지방계약법 (10억 이상) | **40일 이상** |

## 주요 금액 기준

| 항목 | 기준 |
|------|------|
| 예정가격 예비가격 범위 | 국가계약법: ±2%, 지방계약법: ±3% |
| 예비가격 작성 | 15개 중 4개 선택 → 산술평균 |
| 입찰보증금 | 입찰금액의 5/100 이상 |
| 계약보증금 | 계약금액의 10/100 (재난시 5/100) |
| 검사 기간 | 14일 이내 (위기시 7일 이내) |
| 대금 지급 | 청구 후 5일 이내 (위기시 3일 이내) |
| 조달청 대금 지급 | **4근무시간 내** |

## 핵심 암기 포인트

- 발주절차: **9단계**, 업무일 기준 약 **20일 내외** 소요
- 사전규격공개: 일반 **5일**, 긴급 **3일**, SW사업 **10일**
- 예비가격: **15개** 작성 → 입찰참가업체가 **4개** 선택 → 산술평균 = 예정가격
- 입찰보증금: **5/100** 이상
- 계약보증금: **10/100** (재난시 5/100)
- 검사: **14일** 이내 (위기 7일), 불가피 사유 해소 후 **3일** 이내
- 대금 지급: **5일** 이내 (위기 3일), 조달청은 **4근무시간** 내
- 낙찰자 선정은 법률상 **'계약의 예약'** (계약 체결 완료가 아님)

## OX 퀴즈

**Q1.** 사전규격공개 기간은 SW사업의 경우 5일이다. → **X** (SW사업은 10일 이상)

**Q2.** 예정가격은 15개 예비가격 중 입찰참가업체가 가장 많이 선택한 4개의 산술평균이다. → **O**

**Q3.** 계약보증금은 계약금액의 5%가 원칙이다. → **X** (10%가 원칙, 재난시 5%)

**Q4.** 낙찰자 선정은 그 자체로 계약상대자 지위를 부여한다. → **X** (법률상 '계약의 예약'에 해당)

**Q5.** 조달청이 위탁받은 건의 대금 지급은 4근무시간 내에 이루어진다. → **O**

**Q6.** 구매 예상 가격이 5,000만 원 미만인 물품은 사전규격공개 대상에서 제외할 수 있다. → **O**

## 함정 노트

- ⚠️ 국가계약법 vs 지방계약법 입찰공고 기간이 **다르게 적용**됨
- ⚠️ 예정가격 예비가격 범위: 국가계약법 **±2%** vs 지방계약법 **±3%**
- ⚠️ 낙찰자 선정 ≠ 계약 체결: 낙찰자 선정은 **'계약의 예약'**, 계약서 날인(전자서명)으로 계약 성립
