# 제3장 발주대상의 경제성 분석(원가계산)

## 개요

공공조달 원가계산은 발주기관 측면에서는 '예정가격 기초조사서' 작성, 조달업체 측면에서는 '산출내역서' 작성을 위한 업무이다.

## 1. 예정가격 결정 순서

「국가계약법 시행령」 제9조에 따라 다음 순서로 적용:

1. **거래실례가격** — 시중에서 빈번하게 거래, 세금계산서 등으로 증빙 가능
2. **원가계산에 의한 가격** — 충분한 거래실례가격이 없는 경우
3. **표준시장단가** — 100억 원 이상 시설공사에 적용
4. **감정·유사거래실례·견적가격**

## 2. 원가계산 비목 구성

| 비목 | 산정 방법 |
|------|-----------|
| 재료비 | 규격별 재료량 x 단위당 가격 |
| 노무비 | 공종별 노무량 x 노임단가 |
| 경비 | 비목별 경비의 합계액 |
| 일반관리비 | (재료비+노무비+경비) x 일반관리비율 |
| 이윤 | (노무비+경비+일반관리비 - 외주가공비·기술료) x 이윤율 |

```
총원가 = 제조(공사)원가 + 일반관리비 + 이윤
예정가격 = 총원가 + 부가가치세
```

## 3. 이윤율 비교

| 분야 | 이윤율 상한 |
|------|-------------|
| **제조** (SW개발 포함) | **25%** |
| **공사** | **15%** |
| **용역** (SW개발 제외) | **10%** |

## 4. 일반관리비율 (주요)

### 제조업
- 음·식료품: 14% / 섬유·의복·가죽: 8% / 나무·나무제품: 9%
- 종이·인쇄출판: 14% / 제1차 금속: 6% / 그 밖의 물품: 11%

### 용역업
- 시설물관리·경비·청소: 9% / 장비유지보수: 10% / 기타용역: 6%

### 공사업: 5~8%

## 5. 원가계산 시점별 분류

| 유형 | 설명 |
|------|------|
| 예정원가(사전원가) | 경제가치 소비 이전, **원칙** |
| 사후정산원가 | 경제가치 소비 후 실제 비용으로 계산 (정산조건부 계약) |
| 물가변동 | 품목·비목 가격 상승·하락 시 계약금액 조정 |
| 설계변경 | 설계서 누락·오류, 현장 상태 차이 등으로 변경 시 |

## 6. 노무비 관련 노임단가

| 분야 | 노임단가 출처 | 발표 주기 |
|------|---------------|-----------|
| 제조 | 중기중앙회 | - |
| 공사 | 대한건설협회 | 매년 상·하반기 2회 |

## 핵심 암기 포인트

- 예정가격 결정 순서: **거래실례 → 원가계산 → 표준시장단가 → 감정·유사·견적**
- 이윤율: 제조 **25%**, 공사 **15%**, 용역 **10%**
- 일반관리비는 **(재료비+노무비+경비) x 일반관리비율**
- 이윤은 **(노무비+경비+일반관리비) x 이윤율** (기술료·외주가공비 제외)
- 표준시장단가: **100억 원 이상** 시설공사
- 면세업체와 계약 시에도 부가가치세 포함하여 원가계산 실시

## OX 퀴즈

**Q1.** 예정가격 결정 시 원가계산이 거래실례가격보다 우선 적용된다.
→ **X** (거래실례가격이 있는 경우 이를 우선 적용하고, 없는 경우 원가계산을 적용한다)

**Q2.** 용역의 이윤율 상한은 15%이다.
→ **X** (용역의 이윤율 상한은 10%이다. 15%는 공사의 이윤율이다)

**Q3.** 일반관리비 산정 시 접대비와 광고선전비는 제외한다.
→ **O** (제조와 직간접적으로 연관성 없는 비목인 접대비, 광고선전비는 제외한다)

**Q4.** 면세업체가 입찰에 참가할 경우 부가가치세를 제외한 가격으로 투찰한다.
→ **X** (면세업체도 부가가치세를 포함한 가격으로 투찰하되, 낙찰 시 부가가치세를 제외하고 계약을 체결한다)

**Q5.** 정부 원가계산은 일반적으로 사전원가계산을 원칙으로 한다.
→ **O** (일반적으로 사전원가계산을 원칙으로 하며, 정산조건부 계약 등에서는 사후원가계산을 적용한다)

**Q6.** 제조원가계산 시 일부 항목을 일반관리비 또는 이윤 다음 비목으로 계상할 수 있다.
→ **X** (재료비, 노무비, 경비 중 일부를 일반관리비 또는 이윤 다음 비목으로 계상하면 안 된다)

## 함정 노트

⚠️ 이윤 산정 시 기술료와 외주가공비는 **제외**하고 계산한다

⚠️ 간접노무비는 직접노무비를 **초과하여 계상할 수 없다** (공사의 경우 기계화·자동화로 불가피한 경우 증빙자료에 따라 초과 가능)

⚠️ 학술연구용역 인건비 기준단가는 1개월 22일, 참여율 50% 기준 산정 → 전일(8시간) 참여 시 기준단가 2배 적용

⚠️ 공사원가계산 순서는 위계구조의 **역순**: 단가산정 → 단가산출서/일위대가표 → 직접공사비내역서 → 공사원가계산서
