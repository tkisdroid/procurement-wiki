# 제1절 공공조달계획 수립 개요

## 1.1 조달계획 단계의 주요 과업

조달계획 단계에서는 공공조달 수명주기(Life Cycle)에서 수행되어야 할 핵심 절차의 구체적인 실행 방법과 관련한 주요 의사결정이 이루어진다.

### 핵심 과업수행 흐름

| 과업 | 내용 |
|------|------|
| 1. 요구사항 식별 | 수요제기 되는 명확한 조달 요구사항 파악 및 정의 |
| 2. 조달사업팀 구성 | 관련 부서와 담당자로 구성된 팀 구성(TF 형태 등) |
| 3. 조달전략계획 개발 | 조달범위, 구매사양/과업명세서, 주요 관리적 요구사항, 조달방법 결정 |

### 가. 요구사항 파악

- 조직 내에 존재하거나 새롭게 제기되는 조달 수요를 파악하고 식별
- 물품, 용역(서비스), 공사 수요 등 다양한 공공조달 수요를 발굴
- 내부 역량으로 해결 가능한지 vs 외부 자원 조달이 필요한지 결정
- 우선순위를 정하고 예산 수준을 종합적으로 고려
- 통합 구매 가능 여부를 검토하여 구매 효율성 개선

### 나. 조달사업팀 구축

- 조달대상과 우선순위가 결정되면 관련 이해관계자로 팀 구성
- **주요 이해관계자**: 결과물이 필요한 부서/담당자 + 입찰, 계약, 이행관리, 검사, 대금지급 업무 수행 부서/담당자
- 필요에 따라 법률, 회계, 감사 부서 또는 외부 전문가로 확대 구성 가능
- 연간 반복적 일상적 조달은 명시적 팀 구성 없이 조직 프로세스에 내재

### 다. 조달전략계획 개발

조달전략 개발회의에서 논의되어야 할 **13가지 주요 주제**:
1. 프로젝트 범위와 기술사양
2. 프로젝트 결과물
3. 중요한 비즈니스 요구사항
4. 조달 방법
5. 계약 유형
6. 예상 일정
7. 예상 비용과 소요 예산
8. 예상되는 이점
9. 성과 측정(필요한 경우)
10. 대안
11. 주요 고려 사항
12. 제약 조건과 위험관리 사항
13. 조달사업팀 배정과 조달 완료일

#### 1) 조달 범위의 정의

- 조달 요구사항을 명확하게 이해하고 물품·서비스·공사의 특징, 기능, 성능 정의
- 과업명세서(SOW: Statement of Work) 포함 내용: 조달 배경, 달성 목표, 계약 상대자의 업무, 주요 결과물, 수행 일정, 책임자 정보

#### 2) 중요한 관리적 요구사항 식별

11가지 검토 주제:
1. 해당 물품/서비스/공사가 필요한 이유
2. 영향 받는 조직, 개인 또는 기업
3. 필수적으로 충족해야 할 기능, 성능 및 품질
4. 획득에 영향을 미치는 주요 요인
5. 예상 비용 또는 확보 가능 예산
6. 공급 시기와 기간
7. 배송/설치 서비스 필요 여부
8. 내부/외부 승인 필요 여부
9. 구체적인 품질 수준 또는 요구사항

#### 3) 조달 방법 결정

| 단계 | 내용 |
|------|------|
| 가) | 가격과 공급업체의 검증된 계약 정보 확인 (단가계약, 제3자단가계약, MAS 등) |
| 나) | 공공조달 관련 법령과 국제협정 적용 대상 정보 확인 (GPA, FTA 등) |
| 다) | 단독·단일 공급원으로 승인된 조달대상 확인 (수의계약) |
| 라) | 조달대상물 유형별 공통·고유 조달 방법 확인 |
| 마) | 소액구매 |
| 바) | 협력적 구매 정보 확인 (공동계약) |

## 핵심 암기 포인트

- 조달계획 단계 3대 핵심 과업: **요구식별 → 조달사업팀 구성 → 조달전략계획 개발**
- 과업명세서(SOW)에 포함될 사항: 조달 배경, 달성 목표, 계약 상대자 업무, 주요 결과물, 수행 일정, 책임자 정보
- 중요한 관리적 요구사항 식별을 위한 검토 주제: **11가지**
- 조달전략 개발회의 논의 주제: **13가지**
- 소액구매: 구매(신용)카드나 계좌이체 등을 통해 직접 구매 가능
- 공동계약: 서로 다른 수요기관이 동일한 조달대상물에 대해 규모의 경제를 확보하기 위한 방식

## OX 퀴즈

**Q1.** 조달 요구사항이 확정되면 바로 입찰 절차에 진입한다. → **X** (우선순위를 정하고 예산 수준을 종합적으로 고려한 후 적합한 조달 방법을 선택해야 함)

**Q2.** 과업명세서(SOW)에는 수요기관의 책임 사항 관련 정보도 포함되어야 한다. → **O**

**Q3.** 다수공급자계약(MAS)은 물품과 서비스 모두에 적용 가능하다. → **O** (조달사업법 제13조)

**Q4.** 단독·단일 공급원 수의계약은 별도의 입찰 절차 없이 계약을 체결할 수 있다. → **O**

**Q5.** 소액구매는 수의계약보다 절차가 더 복잡하다. → **X** (수의계약보다 더 간소화되어 민간의 일반 상거래와 같은 방법임)

**Q6.** 조달사업팀에는 반드시 외부 전문가가 포함되어야 한다. → **X** (필요에 따라 확대 구성될 수 있으나, 반드시는 아님)

## 함정 노트

- ⚠️ 내부 역량 vs 외부 조달: 조달 요구사항 식별 후 **먼저 내부 역량으로 해결 가능한지를 판단**해야 함
- ⚠️ 과업명세서(SOW) vs 구매사양(Specification): SOW는 요구사항의 세부 설명을 포함하는 문서, 사양은 물리적·기능적 특성을 설명하는 것
- ⚠️ 조달전략계획은 한 번 수립되면 끝이 아니라 **새로운 정보가 수집될 때마다 업데이트**해야 함
