# 제4절 안전관리물자 지정제도

## 개요
국민 안전과 직결되는 조달물자를 지정해 강화된 품질관리를 적용하는 제도이다.

## 분야 및 등급

| 분야 | I등급 | II등급 |
|------|-------|--------|
| 보건위생안전 | 9개 | 12개 |
| 도로안전 | 13개 | 39개 |
| 소방안전 | 15개 | 15개 |
| 공공안전 | 16개 | 33개 |
| 수질안전 | 18개 | 12개 |
| 생활안전 | 27개 | 17개 |
| 수해안전 | 8개 | - |

### I등급 주요 예시
- 보건위생: 살균제, 방진마스크, 보건용마스크, 방역용소독기
- 도로: 스틸그레이팅, 가드레일, 도로반사경, 도로표지병
- 소방: 화재수신기, 연기감지기, 수동식소화기, 공기호흡기
- 공공: 드론, LED투광등, 저출력심장충격기, 구명조끼, 영상감시장치
- 수질: 활성탄, 제수밸브, 정수기, 염소투입기
- 생활: 엘리베이터, 인조잔디, LED실내조명, 공기청정기

## OX 퀴즈

**Q1.** 안전관리물자는 3개 분야로 구분된다.
→ **X** (6개 분야 + 수해안전)

**Q2.** 보건용마스크는 안전관리물자 I등급에 해당한다.
→ **O**

**Q3.** 안전관리물자는 일반 물품과 동일한 품질관리를 적용받는다.
→ **X** (강화된 품질관리 적용)
