# 나라장터 사용자 등록

## 1. 입찰참가자격 등록 개요

입찰참가자격 등록은 입찰에 참여하려는 업체가 필요한 자격 정보를 제출하고, 해당 정보가 입찰 참여 자격요건에 부합하는지를 검증하는 시스템이다.

- **대상**: 공공입찰 참가를 통해 수요기관에 물품·공사·용역을 공급하려는 사업자
- **관련 규정**: 「국가종합전자조달시스템 입찰참가자격 등록규정」
- **시스템**: 나라장터(www.g2b.go.kr)

## 2. 등록 절차

### 전체 흐름

```
개인회원 가입(본인인증) → 이용약관 동의 → 입찰참가자격 등록신청
→ 온라인 서류 제출 → 등록신청 현황 조회 → 승인 또는 반려
```

### 등록신청 메뉴 진입

나라장터에 로그인 후 '입찰참가자격 등록신청' 메뉴로 진입하면 다음 화면이 표시된다.

![입찰참가자격 등록신청 메뉴 진입화면](/procurement-wiki/images/textbook/book4/p10_fig1.png)

> 📌 **실무 포인트**: 이 화면의 메뉴 위치(이용자 관리→조달업체)는 자주 출제된다. 등록 전 사업자등록증·인감 스캔 파일을 미리 준비해야 한다.

### 로그인·신규등록 선택

기존 회원은 로그인, 신규는 '신규 등록' 버튼을 선택하여 진입한다.

![입찰참가자격 등록신청(로그인·신규등록 선택)](/procurement-wiki/images/textbook/book4/p10_fig2.png)

> 📌 **실무 포인트**: 신규 등록 클릭 후 본인인증부터 시작된다. 반려된 신청 건을 다시 진행하려면 '반려 건 불러오기'를 사용하면 처음부터 다시 입력할 필요가 없다.

### 이용약관 동의

등록신청 진행 전 이용약관과 청렴서약서에 동의해야 한다.

![입찰참가자격 등록신청 이용약관 동의](/procurement-wiki/images/textbook/book4/p11_fig1.jpeg)

> 📌 **실무 포인트**: 이용약관 동의 → 등록신청 진행 중 취소하면 이용약관 동의 절차를 **다시** 진행해야 한다는 함정이 출제된다.

### 기본 신청화면

이용약관 동의 후 기본 신청화면에서 사업자등록번호를 입력한다.

![입찰참가자격 등록신청 기본 신청화면](/procurement-wiki/images/textbook/book4/p11_fig2.png)

> 📌 **실무 포인트**: 사업자등록번호와 법인등록번호의 **중복 등록 여부 의무 확인** 절차가 필수. 중복 시 등록이 불가하다.

### 세부 단계 (4단계)

| 단계 | 내용 | 비고 |
|------|------|------|
| 1단계 | 등록신청 | 기본정보, 대표자정보, 조달대상물 선택 |
| 2단계 | 신청 확인 및 안내문 인쇄 | 제출서류 유무 확인 |
| 3단계 | 온라인 서류 제출 | 20일 이내 미제출 시 반려 |
| 4단계 | 검토 및 승인 | 관할 지방조달청에서 검토·승인 |

### 필수 입력 정보

**기본 정보:**
- 사업자등록번호, 상호, 주소, 전화번호, 개업일자, 종업원 수, 법인등록번호
- 사업자등록번호와 법인등록번호는 **중복 등록 여부 의무 확인**

![입찰참가자격 등록신청(기본 정보 등록)](/procurement-wiki/images/textbook/book4/p12_fig1.png)

> 📌 **실무 포인트**: 상호·주소·전화번호 등은 사업자등록증 기재내용과 일치해야 한다. 변경 시 즉시 변경 등록을 해야 입찰자격 무효화를 피할 수 있다.

**대표자 정보:**
- 대표자 성명, 주민등록번호 (실명인증 필수)
- 대표자 2명 이상 시 **'대표 대표자' 1명 지정** 필수

![입찰참가자격 등록신청(대표자 정보 입력)](/procurement-wiki/images/textbook/book4/p12_fig2.png)

> 📌 **실무 포인트**: 대표자 2명 이상이면 반드시 '대표 대표자' 1명을 지정해야 한다는 점이 함정 출제. 누락 시 등록 진행 불가.

**조달대상물 등록:**
- 공급물품, 제조물품, 공사·용역·기타업종 중 1개 이상 선택

## 3. 공급물품 vs 제조물품 비교

| 구분 | 공급물품 | 제조물품 |
|------|----------|----------|
| 정의 | 타사 생산 물품을 구매하여 공급 | 직접 생산·제조하여 공급 |
| 등록 방법 | '세부품명 찾기'로 10자리 세부품명번호 선택 | 세부품명 조회 후 추가 |
| 추가 요건 | 없음 | 직접생산확인기준 충족 증명 서류 제출 |
| 직접생산 확인 | 불필요 | 조달청 또는 중소벤처기업부 기준 적용 |

> **참고**: 2024년 2월부터 제조물품 등록신청 단계에서 '제조공정표' 제출 면제. 납품 실적 발생 후 직접생산 확인 실태조사 시 제출.

## 4. 용역·공사·기타업종 등록

- 해당 업종 소관 정부부처 법령에 따른 자격·면허·인증 요건 충족 필수
- '업종사항 찾기'로 해당 업종 검색 후 면허등록번호 입력
- 물품으로 등록한 업체도 요건 충족 시 용역·공사 업체로 **중복 등록 가능**

### 등록기관(지방조달청) 선택

사업장 소재지 기준으로 관할 지방조달청을 선택한다.

![입찰참가자격 등록신청(지방청 선택)](/procurement-wiki/images/textbook/book4/p13_fig1.jpeg)

> 📌 **실무 포인트**: 지방청 선택은 사업장 소재지 기준이라는 점이 함정. 본사·사업장 주소가 다르면 사업장 주소가 기준이 된다.

### 인증서 전자서명

마지막 단계에서 인증서 전자서명으로 등록신청 정보를 전송한다.

![입찰참가자격 등록신청 인증서 전자서명](/procurement-wiki/images/textbook/book4/p13_fig2.jpeg)

> 📌 **실무 포인트**: 전자서명용 인증서는 사업자등록번호·법인등록번호와 일치해야 한다. 개인 인증서로는 사업자 등록 불가. 청렴서약서 동의도 함께 진행된다.

## 5. 승인 완료 후

- '경쟁입찰참가자격등록증' 출력 가능
- 나라장터를 통한 공공조달 입찰과 계약 체결 자격 확보

## 핵심 암기 포인트

- 최종 승인은 사업장 소재지 **관할 지방조달청**에서 결정
- 등록신청 후 **20일 이내** 서류 미제출 시 신청 반려
- 기본정보·대표자 정보의 변경 미반영 시 **입찰참가자격 무효화**
- 전자서명(인증서)을 통해 등록신청 정보 전송
- 사전동의서와 **청렴계약서(서약서)** 동의 필수

## OX 퀴즈

**Q1.** 입찰참가자격 등록은 조달업체가 등록한 사항에 대해 조달청 본청에서 직접 승인한다.
→ **X** (관할 지방조달청에서 신청 내용 및 제출서류를 검토한 후 승인한다)

**Q2.** 개인사업자도 나라장터에 입찰참가자격을 등록할 수 있다.
→ **O** (법인뿐 아니라 개인사업자도 등록 가능하다)

**Q3.** 중소기업자 간 경쟁제품에 해당하는 제조물품은 등록신청 단계에서 직접생산 확인 증명서를 제출해야 한다.
→ **O** (중소벤처기업부 장관이 지정하는 중소기업자 간 경쟁제품의 경우 '직접생산 확인 증명서'를 등록신청 단계에서 제출해야 한다)

**Q4.** 등록신청이 반려된 경우 처음부터 새로 신청해야 한다.
→ **X** (반려신청 건의 기존 등록신청 정보를 불러와 다시 신청 절차를 진행할 수 있다)

**Q5.** 복수의 물품 공급이 가능한 경우 해당하는 복수의 세부품명을 등록할 수 있다.
→ **O** (복수의 세부품명을 조회하여 추가 등록 가능하다)

## 함정 노트

⚠️ 세부품명 선택 시 실제 공급 가능한 물품과 상품 정보가 일치하는지 정확히 확인해야 한다 -- 부정확한 선택은 입찰참가자격 문제 또는 납품 문제 발생

⚠️ 등록기관(지방조달청) 선택은 사업장 소재지 기준으로 해야 한다

⚠️ 이용약관 동의 → 등록신청 진행 중 취소하면 이용약관 동의 절차를 다시 진행해야 한다
