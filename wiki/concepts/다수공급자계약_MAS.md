# 제2절 다수공급자계약(MAS) 절차 및 이행관리

## 2.1 MAS 개요

- 품질·성능·효율이 같거나 비슷한 수요물자를 **2인 이상** 계약상대자와 단가계약
- 수요기관이 **선택** 구매
- 계약기간: **3년** 원칙

### 일반 단가계약과 MAS 비교

| 구분 | 일반 단가계약 | MAS |
|------|-------------|-----|
| 계약상대자 | 1인 | 2인 이상 |
| 낙찰 방식 | 경쟁 후 1인 확정 | 적격성평가+가격협상 |
| 구매 흐름 | 계약→납품 | 등록→선택→경쟁→납품 |
| 진입 장벽 | 높음 | 낮음 (계약 이후) |
| 계약 후 경쟁 | 없음 | 있음 (2단계 경쟁) |

## 2.2 MAS 계약 절차

1. **적격성평가**: 신용등급 B- 미만, 부도·파산 → 결격
2. **가격협상**: 최근 1년 규격별 가격자료 기반 협상기준가격
3. **계약 체결**: 3년 원칙
4. **납품요구**: 기준금액 이상 시 2단계 경쟁
5. **계약관리**: 90일 경과 + 3% 이상 변동 시 금액 조정

## 2.3 MAS 2단계경쟁

### 대상 기준금액

| 구분 | 기준금액 |
|------|---------|
| 용역 (중소기업자 간 경쟁/중소기업 공급) | **1억 원** 이상 |
| 용역 (기타) | **5천만 원** 이상 |
| 물품 (중소기업자 간 경쟁) | **1억 원** 이상 |
| 물품 (일반) | **5천만 원** 이상 |
| 물품 **5억 원 이상** | 제안공고 실시 |

### 2단계경쟁 예외
- 재해복구·방역 긴급 구매
- 수요기관장 명백한 사유로 예외 요청

### 2단계경쟁 회피 금지
- 기준금액 미만으로 **분할 납품요구 금지**
- 최근 **30일** 이내 동일업체·동일 세부품명 합산 검증

### 제안요청
- **5인** 이상 계약상대자 대상
- 시스템 자동 추천 2인 추가 가능
- 15인 이상: 무작위 추천 10인 중 선정
- 제안 마감: 5천만 미만 만 **3일** 이상, 5천만 이상 만 **5일** 이상
- 중소기업자 간 경쟁제품: 계약가격의 **90%** 미만 제안 불가

### 납품업체 선정 (종합평가)

| 구분 | 항목 | 배점 |
|------|------|------|
| 기본평가 (50점 이상) | 가격 20점↑, 납품실적 20점↑, 경영상태 10점↓, 신인도 ±1.75 | |
| 선택평가 (50점 이하) | 적기납품, 서비스수행능력, 선호도, 사후관리, 지역업체 등 | |

- 합산점수 동점 → 사후관리 점수 높은 자 → 제안가격 낮은 자

## OX 퀴즈

**Q1.** MAS 계약기간은 5년이 원칙이다.
→ **X** (3년이 원칙이다)

**Q2.** MAS에서 중소기업자 간 경쟁제품 용역의 2단계경쟁 기준금액은 1억 원이다.
→ **O**

**Q3.** MAS 2단계경쟁 시 3인 이상의 계약상대자에게 제안요청하면 된다.
→ **X** (5인 이상이다)

**Q4.** 중소기업자 간 경쟁제품 MAS에서 계약가격의 80% 미만으로 제안할 수 있다.
→ **X** (90% 미만 제안 불가)

**Q5.** 2단계경쟁 회피 판단 시 최근 60일 이내 동일업체 납품요구를 검증한다.
→ **X** (최근 30일 이내이다)

**Q6.** MAS 2단계경쟁에서 제안 마감 전까지 제안을 취소할 수 있다.
→ **O** (마감 후에는 취소 불가)

## 함정 노트

⚠️ MAS 2단계경쟁 기준금액: 중소기업 **1억** / 기타 **5천만** — 물품과 용역 동일.

⚠️ 5인 이상 제안요청 필수 (2~5인 미만은 불가피한 사유 시 가능 + 사유 입력 필요).

⚠️ 2단계경쟁 **회피 금지**: 분할 납품요구로 기준금액 이하로 만드는 행위 차단 → 30일 이내 합산 검증.

⚠️ 90% 하한 제한은 **중소기업자 간 경쟁제품**에만 적용된다.
