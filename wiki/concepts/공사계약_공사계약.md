# 제3절 공사계약

## 3.1 건설 공사비

### 공사비 종류 (생애주기별)

| 단계 | 명칭 | 설명 |
|------|------|------|
| 기획 | 추정가격 | 개략견적 (단위면적·길이당 공사비) |
| 설계 | 설계가격 | 실시설계 완료 후 명세견적 |
| 구매 직전 | 조사가격 | 발주기관이 설계가격 검증 |
| 입찰 | 기초금액 | 추정가격 + 부가세, 복수예비가격 기준 |
| 입찰 | 복수예비가격 | 기초금액 ±2%(국가)/±3%(지방) 범위 **15개** |
| 입찰 | 예정가격 | 가장 많이 선정된 **4개** 산술평균 |
| 계약 | 계약금액 | 낙찰자 결정 후 확정 |

### 예정가격 결정 기준
1. **거래실례가격**: 조달청 가격정보, 전문기관 공표가격, 직접조사 가격, 법령 결정 가격
2. **원가계산 가격**: 직접공사비 + 간접공사비 + 일반관리비 + 이윤 + 부가세
3. **표준시장단가**: 축적된 공사비 활용

### 원가구성 체계

```
직접공사비 (재료비+노무비+경비)
+ 간접공사비 (간접노무비+현장경비)
= 순공사원가(순공사비)
+ 일반관리비
+ 이윤
= 총원가
+ 부가가치세
= 건설공사 금액
```

## 3.2 공사계약 입찰·낙찰

### 입찰참가자격 사전심사(PQ)
- 대상: 추정가격 **300억** 이상 공사
- 평가: 시공경험, 기술능력, 경영상태, 시공평가결과, 신인도

### 종합심사낙찰제
- 공사수행능력 **50점** + 입찰금액 **50점** = 100점
- 사회적 책임 가점 최대 2점
- 가격 산출 적정성: 단가 감점(-4), 하도급 감점(-2)

### 적격심사 (300억 미만)
- 공사도 85점 이상 통과

## 3.3 계약 체결

### 계약서 작성 생략 가능
- 계약금액 **3,000만 원** 이하
- 국가기관-지방자치단체 상호 계약
- 성격상 불필요한 경우

### 계약보증금
- 계약금액 **10%** 이상 (공사이행보증서 40%)
- 5,000만 원 이하: 면제 가능
- 위기 시: 5% 이상으로 감액 가능

## OX 퀴즈

**Q1.** 복수예비가격은 기초금액의 ±5% 범위에서 15개를 작성한다.
→ **X** (국가계약법 ±2%, 지방계약법 ±3%)

**Q2.** 예정가격은 가장 많이 선정된 4개 복수예비가격의 산술평균이다.
→ **O**

**Q3.** 입찰참가자격 사전심사(PQ)는 추정가격 100억 원 이상 공사에 적용된다.
→ **X** (300억 원 이상이다)

**Q4.** 종합심사낙찰제에서 입찰금액 배점은 60점이다.
→ **X** (50점이다)

**Q5.** 계약금액 5,000만 원 이하인 경우 계약서 작성을 생략할 수 있다.
→ **X** (3,000만 원 이하이다)

**Q6.** 순공사원가는 직접공사비와 간접공사비의 합이다.
→ **O**

## 함정 노트

⚠️ 복수예비가격 범위: 국가 **±2%** / 지방 **±3%** — 혼동 주의.

⚠️ PQ 대상: **300억** 이상 (100억, 500억 등으로 변형 출제).

⚠️ 계약서 생략: **3,000만 원** 이하 (5,000만 원은 계약보증금 면제 기준).

⚠️ 종합심사 = 50:50 배점 구조 (공사수행능력:입찰금액).
