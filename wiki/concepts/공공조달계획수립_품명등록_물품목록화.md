# 공급대상물 유형별 품명 등록(물품목록화)

## 1. 목록화 개요

납품 가능한 물품의 상품 속성과 일치하는 세부품명이 존재하지 않는 경우, **물품목록화 요청**을 통해 새로운 세부품명을 만들어야 한다. 목록화 절차는 **목록정보시스템**을 통해 수행된다.

### 입찰참가자격 등록 시 세부품명 선택

공급물품과 제조물품 모두 등록 시 세부품명을 선택해야 한다. 공급물품은 10자리 세부품명번호를 검색하여 선택한다.

![입찰참가자격 등록신청(공급물품 세부품명 선택)](/procurement-wiki/images/textbook/book4/p18_fig1.jpeg)

> **실무 포인트**: 부정확한 세부품명 선택은 입찰참가자격 문제뿐 아니라 납품 단계 문제로도 이어진다. 실제 공급 가능한 물품과 일치 여부 확인 필수.

제조물품 등록 시에도 동일한 세부품명 검색 화면이 사용되지만, 직접생산확인 증명서가 추가로 요구된다.

![입찰참가자격 등록신청(제조물품 세부품명 선택)](/procurement-wiki/images/textbook/book4/p18_fig2.jpeg)

> **실무 포인트**: 중소기업자 간 경쟁제품 해당 시 직접생산확인서를 등록 단계에서 제출 필수. 2024년 2월부터 제조공정표는 면제되어 납품 후 실태조사 시 제출.

## 2. 물품분류체계

| 단계 | 분류 | 번호 자릿수 |
|------|------|-------------|
| 1단계 | 대분류 | 2자리 |
| 2단계 | 중분류 | 4자리 |
| 3단계 | 소분류 | 6자리 |
| 4단계 | 세분류 | 8자리 |

**물품목록번호** = 품명번호(8자리) + 식별번호(8자리) = **16자리**

## 3. 목록화 요청 방법

### 개별요청 (3가지 세부 방법)
1. **요청번호 참조등록**: 기존 요청 정보를 불러와 수정하여 등록
2. **물품식별번호 참조등록**: 기존 식별번호 정보를 활용하여 등록
3. **세부품명 검색 등록**: 직접 세부품명 검색 후 요청 내용 입력

### 일괄요청
- 동일 세부품명의 품목이 **10개 이상**인 경우 활용
- 엑셀 양식에 정보 입력 후 업로드 (최대 500건)
- 양식 임의 변경 시 데이터 검증 오류 발생

## 4. 목록화 처리 소요 기간

| 유형 | 소요 기간 |
|------|-----------|
| 품명 등록 | 24일 이내 |
| 신산업 상품 | 17일 이내 |
| 품목 등록 | 8일 이내 |
| 품목 변경 | 5일 이내 |

## 5. 품목 구분 (5종)

| 구분 | 용도 |
|------|------|
| 일반용 | 일반적 물품관리 + 계약 용도 모두 활용 |
| 계약용 | 계약 진행에만 활용 (일반 물품관리 불가) |
| 시설자재용 | 원가계산 용도로만 활용 |
| 입찰공고용 | 특정 계약업무 처리에만 활용 |
| 물품관리용 | 일반적 물품관리 용도만 활용 (계약업무 불가) |

## 6. 품목 등록 필수 정보

- 물품분류번호, 세부품명, 상품원산지국가명
- 주문자상표부착(OEM) 상품 여부, 부품 여부, 복합상품 여부
- 전통문화상품 여부, 제조업체명, 모델명, 단위
- 이미지(실물 사진 필수, 그래픽 이미지 불가)
- 규격서, 도면, 매뉴얼 등 첨부파일

## 7. 융복합상품

| 유형 | 정의 | 입력 요건 |
|------|------|-----------|
| 융합상품 | 2개 이상 세부품명이 결합하여 하나의 몸체를 이루는 상품 | 2개 이상 세부품명 입력 |
| 복합상품 | 2개 이상 세부품명이 복합되어 새로운 기능을 구현하는 상품 | 2개 이상 식별번호와 세부품명 입력 |

> 융복합상품을 제조물품으로 등록 시, 구성품 중 1개 이상이 경쟁입찰참가자격증에 제조물품으로 등록되어 있어야 한다. OEM 상품은 공급물품으로 등록해야 한다.

## 핵심 암기 포인트

- 동일한 품명이나 품목이 이미 등록되어 있으면 목록화 요청 불가 → **사전 확인 필수**
- 개별요청은 품목 수에 관계없이, 일괄요청은 **10개 이상**인 경우에만 가능
- 조달청 보완 요청 시 **15일 이내** 보완 미제출 시 자동 반려
- 품목명 생성 규칙: **세부품명-제조업체명-모델명-대표속성값**
- 일괄등록 엑셀 양식 임의 변경 불가 (함수·수식 적용 양식)

## OX 퀴즈

**Q1.** 개별요청은 10개 미만의 품목에만 활용할 수 있다.
→ **X** (개별요청은 품목 수에 관계없이 활용 가능하며, 일괄요청이 10개 이상인 경우에만 활용 가능한 것이다)

**Q2.** 목록화 요청 시 등록하고자 하는 물품과 동일한 품명이 이미 등록되어 있으면 진행할 수 없다.
→ **O** (동일한 품명이나 품목이 등록되어 있지 않아야 진행 가능하다)

**Q3.** 품목 등록 시 이미지는 그래픽 이미지로 등록해도 된다.
→ **X** (실물 사진을 등록해야 하며 그래픽 이미지는 등록할 수 없다)

**Q4.** 일괄등록 양식의 항목을 필요에 따라 추가할 수 있다.
→ **X** (양식을 임의로 변경하거나 항목을 추가하면 오류가 발생한다)

**Q5.** 활용 용도 선택 시 등록 이후 다른 용도로 활용할 수 없다.
→ **X** (활용 용도는 참고사항으로, 등록 이후 다른 용도로 활용하는 것도 허용된다)

## 함정 노트

⚠️ 수정 시 행정 소요 일수가 다시 계산되므로 후속 일정을 고려해야 한다

⚠️ 복합물품으로 구성된 시스템장비의 물품식별번호는 물품관리에 사용할 수 없으므로, 구성 품목 각각에 별도 물품식별번호를 부여하여 목록화 요청해야 한다

⚠️ 외국산 물품의 경우 일괄등록 시 '제조업체 카탈로그' 첨부 필수, OEM 상품의 경우 '상표권이 명시된 제조계약서' 첨부 필수
