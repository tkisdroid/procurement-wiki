# 제1절 효과적 계약관리 계획

## 1.1 계약관리 원칙

대법원은 공공계약의 법적 성격을 **"사적자치가 적용되는 사법(私法)계약"**으로 본다 (대법원 2017.12.21. 선고 2012다74076).

### 계약의 자유

| 자유 유형 | 내용 |
|-----------|------|
| 체결의 자유 | 청약·승낙의 자유 |
| 상대방 선택의 자유 | 원하는 상대와 계약, 특정인 강요 불가 |
| 내용 결정의 자유 | 가장 중요한 요소, 변경·보충 포함 |
| 방식의 자유 | 특정 방식 불요, 합의로 성립 |

계약의 자유에는 **계약을 체결하지 않을 자유**도 포함된다.

### 국가계약법 제5조 (계약의 원칙)

- ①항: **서로 대등한 입장**에서 합의, **신의성실**의 원칙에 따라 이행
- ②항: 국제입찰 시 호혜의 원칙 (WTO 정부조달협정 가입국 차별 금지)
- ③항: 계약상대자의 이익을 **부당하게 제한하는 특약(부당특약)** 금지
- ④항: 부당특약은 **무효**

## 1.2 계약관리요소

### 계약의 기능 (5가지)

1. **증거기능**: 합의의 증거, 이견 해결 기준
2. **관리기능**: 작업자 행동·내용 촉진, 계약이행 관리
3. **위험전가기능**: 위험 부담 주체를 계약서에 명시
4. **지불기능**: 대가 지급 조건 포함 (확정가격계약, 원가보상계약 등)
5. **동기부여기능**: 이익 창출 의지, 유인계약(IC)

### 계약의 성립요건 (4가지)

1. **의사표시**: 쌍무계약(합의계약)
2. **이행가능성**: 기간·장소·수량·가격 충족, 계약능력 보유
3. **법적·사회적 타당성**: 반사회적·반법률적 내용 불가
4. **확정성**: 분명한 내용, 제3자도 인지 가능

### 계약서 해석 우선순위

1. **특수조건** > 일반조건
2. **수기·별도 명시 문구** > 기본 문서
3. **강행규정** > 일반규정
4. **계약서 명시사항** > 민법 일반원칙 (단, 민법 위배 시 해당 조항 배제)

#### 조달청 물품구매계약 해석 순위 (제27조)
1. 계약서(갑·을지)
2. 물품구매(제조)계약 추가특수조건
3. 물품구매계약 품질관리 특수조건
4. 물품구매(제조)계약 특수조건
5. 물품구매 규격서(시방서·보완규격)
6. 물품구매(제조)계약 일반조건
7. 물품구매(제조)입찰권유서
8. 물품구매(제조)입찰유의서

## 1.3 계약 체결 후 발주기관과 착수 협의

- **용역**: 계약문서에 따라 착수, 착수신고서 제출 (공정예정표, 인력·장비투입계획서 등)
- **공사**: 착공신고서 제출 (현장기술자지정신고서, 공정예정표, 안전·환경·품질관리계획서 등)

## 1.4 착수 시점과 착공 기한

| 구분 | 기한 |
|------|------|
| 용역 | 계약 체결 후 **14일** 이내 착수 |
| 공사 (10억 미만) | 계약 체결일로부터 **10일** 이전 착공 불가 |
| 공사 (10억 이상) | 계약 체결일로부터 **20일** 이전 착공 불가 |
| 공사기간 30일 미만 | 착공신고서 생략 가능 |

- 공사: 월별 공정보고 → 익월 **14일까지** 발주기관 제출
- 산출내역서: 추정가격 **100억 이상** 공사 시 입찰서에 첨부 의무

## 1.5 하도급계약 승인과 변경

- 하도급계약 통보: 체결 후 **30일** 이내
- 공사감독자: 적정성 검토 후 **7일** 이내 발주청 보고
- 하도급 적정성 심사 대상:
  - 하도급금액 < 도급금액 중 하도급 부분의 **82%**
  - 하도급금액 < 발주자 예정가격의 **64%**

## 함정 노트

⚠️ "공공계약 = 공법상 계약"으로 오인하기 쉬우나, 대법원은 **사법상 계약**으로 본다.

⚠️ 부당특약의 효력: "취소할 수 있다"가 아니라 **"무효"**이다 (국가계약법 §5④).

⚠️ 착공 기한: "10일/20일 **이내** 착공"이 아니라 "10일/20일 **이전에 착공 불가**"이다 (최소 유예기간 개념).

⚠️ 하도급 심사 기준 82%와 64%를 혼동하지 말 것: 82%는 도급금액 대비, 64%는 예정가격 대비이다.
