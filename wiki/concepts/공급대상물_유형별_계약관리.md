# 제6장 공급 대상물 유형별 계약 관리

## 하위 절

| 절 | 제목 | 핵심 내용 |
|---|------|----------|
| 제1절 | 공통 조달관리 절차 수행 | 전자계약, 착공/착수, 검사검수, 대금관리 |
| 제2절 | 물품계약 입찰 절차 수행 | 입찰공고조회, 투찰, 적격심사, 낙찰 |
| 제3절 | 용역계약 입찰 절차 수행 | 사업설명회, 실적심사, 공동수급협정서 |
| 제4절 | 공사계약 특화 절차 수행 | PQ심사, 적격성심사, 종합심사, 종합평가 |

## 1. 공통 이행 절차 (물품·용역)

### 이행 세부업무 (8개)
네트워크론, 착공(착수), 신용장(선적), 검사(검수), 대금관리, 이행점검, 종결, 환수(환불), 종결관리

### 검사(검수) 절차
```
조달업체: 검사신청 → 발주기관: 검사 실시 → 합격 시 물납영수증 발급
→ 조달업체: 대금청구서 작성 → 발주기관: 지출결의·지급
```

### 대금관리
- 공동계약: **대표업체만** 대금청구서 작성 가능
- 구성업체: 대금청구 **사실확인서** 작성으로 자사 지분 직접 지급
- 세금계산서: 국세청 시스템과 연계하여 온라인 발급
- 유보금: 시운전·최종검사 등 조건 충족 시 해제 청구 가능

## 2. 물품계약 입찰 절차

### 입찰서 제출
- 투찰 제한사항·부정당업자 제재 여부 확인
- 입찰서류 제출 → 가격입찰서 제출 (총액 입력)
- **복수 예비가격 15개** 중 2개 추첨번호 선택
- 법인사업자: **법인용 공동인증서** 전자서명 필수
- 제출 완료 후 **송수신 상세 이력** 확인 (전송 실패 시 재투찰)

### 적격심사
- 심사항목: 납품실적, 기술능력, 경영상태, 신인도
- 신인도 가산: 혁신형중소기업, 신규고용 우수기업, 여성·장애인·청년 고용률 등
- 신인도 감점: 납품지연(지체상금), 하도급 위반, 임금체불 등
- 심사 결과에 이의가 있으면 **이의신청** 가능

## 3. 용역계약 차별적 절차

- **사업(과업)설명회**: 현장설명회 참가 신청·조회
- **실적심사**: 실적심사신청서 작성·조회
- **공동수급협정서**: 공동수급체 구성 시 협정서 등록·확인
- 기본 절차는 물품계약과 유사 (동일 나라장터 기능 활용)

## 4. 공사계약 특화 절차 (5개 부문)

### PQ심사 (사전자격심사)
- 시공 경험, 기술 능력, 경영 상태를 **사전** 평가
- 통과 업체만 입찰참가 허용
- 평가 분야: 경영상태, 시공경험, 기술능력, 신인도(표준계약), 신인도(일자리창출)

### 적격성심사
- 부실공사 방지, 공공사업 품질 향상 목적
- 해당 계약 성공적 이행 능력 종합 평가
- 심사 결과 이의신청 가능

### 종합심사낙찰제 (국가계약법)
- 종합심사신청서 제출 → 서류제출 대상자 선정 → 심사서류 제출 → 낙찰자 선정

### 종합평가낙찰제 (지방계약법)
- 종합심사낙찰제와 기본적으로 동일·유사
- 종합평가신청서 제출 → 심사 → 낙찰자 선정

### 국가유산수리공사
- 일반공사와 별도 구분
- 종합심사 또는 종합평가 적용
- 부실 수리 예방, 적정 대가 보장, 수리품질 향상 목적
- 2017년부터 본격 시행

### 공사계약 검사(검수) 특이사항
- **기성청구**: 공정별 완성부분 대가 청구 (기성확인요청서 → 기성확인서)
- **준공확인**: 최종 완공 시 잔여 대금 청구 (준공확인요청서 → 준공확인서)

## 핵심 암기 포인트

- 물품·용역 전자계약: **초안→응답서→통보서** 3단계
- 물품 투찰 시 복수 예비가격 **15개 중 2개** 추첨번호 선택
- 공동계약 대금청구: **대표업체만** 가능, 구성업체는 확인서
- PQ심사: **사전** 평가 → 통과 업체만 입찰참가
- 종합심사(국가계약법) vs 종합평가(지방계약법)
- 국가유산수리공사: 2017년부터 종합심사 본격 시행

## OX 퀴즈

**Q1.** 공동계약의 구성업체도 직접 대금청구서를 작성할 수 있다.
→ **X** (대표업체만 대금청구서를 작성할 수 있으며, 구성업체는 대금청구 사실확인서를 작성한다)

**Q2.** PQ심사를 통과하지 못한 업체도 입찰서 제출은 가능하다.
→ **X** (PQ심사를 통과한 조달업체에만 입찰참가가 허용된다)

**Q3.** 종합심사낙찰제는 지방계약법에서 종합평가낙찰제로 명명된다.
→ **O** (지방계약법 적용 공사에서는 종합평가낙찰제라 하며 기본 절차는 동일·유사하다)

**Q4.** 물품 적격심사에서 신인도 항목은 가산점만 부여된다.
→ **X** (납품지연, 하도급 위반, 임금체불 등 감점 항목도 존재한다)

**Q5.** 입찰서 제출 후 전송 결과를 확인할 필요가 없다.
→ **X** (송수신 상세 이력을 조회하여 전송 실패 시 재투찰해야 한다)

## 함정 노트

⚠️ 전자입찰서 제출 임시 허용: 자기정보가 사실과 다르게 등록되어 제출이 제한된 자가 **긴급히** 입찰 참가해야 하는 경우 '임시 허용 신청서' 제출 가능

⚠️ 제안서 제출 완료 후 반드시 **목차조견표**와 **평가항목조견표**를 등록해야 한다

⚠️ 공사계약 기성청구 vs 준공확인: **기성 → 공정별 부분 대가**, **준공 → 최종 완공 잔여 대금**
