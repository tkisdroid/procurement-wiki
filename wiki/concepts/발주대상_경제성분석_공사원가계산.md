# 공사원가계산

「예정가격작성기준」 제15조에 따라 시공과정에서 발생한 재료비, 노무비, 경비의 합계액으로 계산한다.

## 1. 공사원가 구성체계

```
공사비 = 총공사원가 + 부가가치세
총공사원가 = 순공사원가 + 본사 공사원가(일반관리비 + 이윤)
순공사원가 = 직접공사비 + 간접공사비
```

### 공사원가 구성체계 다이어그램

공사비는 총공사원가와 부가가치세, 총공사원가는 순공사원가와 본사 공사원가(일반관리비+이윤)로 구성된다.

![공사원가 구성체계(공사비→총공사원가→순공사원가)](/procurement-wiki/images/textbook/book4/p119_fig1.png)

> 📌 **실무 포인트**: 공사비-총공사원가-순공사원가-직접/간접공사비의 위계 구조는 핵심. 각 비목의 포함 관계를 정확히 외워야 한다.

### 철근콘크리트 구조물 설치 공사원가 사례

실제 공종(철근콘크리트 구조물 설치)을 예시로 한 공사원가 구성 사례이다.

![철근콘크리트 구조물 설치 공사원가 구성체계](/procurement-wiki/images/textbook/book4/p121_fig1.png)

> 📌 **실무 포인트**: 공종별로 직접재료비·노무비·경비가 어떻게 산정되는지 보여주는 사례. 표준품셈 적용 방법 확인.

## 2. 직접공사비

| 비목 | 내용 |
|------|------|
| 재료비 | 직접재료비(주요재료비 + 부분품비) + 간접재료비(소모재료비 + 가설재료비) |
| 직접노무비 | 공사현장 직접 종사 노무자의 노동력 대가 |
| 산출경비 | 기계손료, 운반비, 가설비, 외주가공비, 시운전비, 환경보전비 등 |

### 직접재료비 산정 방법

공사 직접재료비는 주요재료비와 부분품비로 구분하여 산정한다.

![직접재료비 산정방법](/procurement-wiki/images/textbook/book4/p123_fig1.png)

> 📌 **실무 포인트**: 재료 구입 **과정**의 부대비용은 재료비, 구입 **후** 부대비용은 경비라는 함정 출제. 작업설·부산물은 재료비에서 공제하지만 기존시설 철거 발생분은 공제하지 않는다.

### 직접노무비 계상방법

공사 직접노무비는 노무량(품셈)과 시중노임단가(대한건설협회)로 산출한다.

![공사원가 직접노무비 계상방법](/procurement-wiki/images/textbook/book4/p125_fig1.png)

> 📌 **실무 포인트**: 노무량은 **표준품셈**, 노임단가는 **대한건설협회**(매년 상·하반기 2회)에서 적용. 간접노무비는 직접노무비를 초과 계상 불가(기계화·자동화 예외).

## 3. 간접공사비

- 4대보험료 (연금, 건강, 고용, 산재보험료)
- 퇴직공제부금비, 산업안전보건관리비
- 환경보전비, 품질관리비, 안전관리비, 폐기물처리비
- 법령 수수료 (공사이행보증서, 건설하도급대금지급보증서 등)

> **주의**: 계약이행보증서와 하자이행보증수수료는 경비에 포함되지 않음

## 4. 공사원가계산 순서

위계구조의 **역순**으로 수행:

```
단가 산정 → 단가산출서/일위대가표 작성 → 직접공사비내역서 → 공사원가계산서
```

| 문서 | 용도 |
|------|------|
| 단가산출서 | 복잡한 공종, 시장가격·노임정보 직접 계산 |
| 일위대가표 | 단순·표준화된 공정의 공사내역 산출 |
| 직접공사비내역서 | 공종별 상세 재료비·노무비·경비 집계 |
| 공사원가계산서 | 직접·간접공사비 총괄 |

### 단가산출서 vs 일위대가표 비교

복잡한 공종은 단가산출서, 단순·표준화된 공정은 일위대가표로 처리한다.

![단가산출서와 일위대가표 비교](/procurement-wiki/images/textbook/book4/p128_fig1.png)

> 📌 **실무 포인트**: 두 문서의 용도 구분이 출제. 단가산출서는 시장가격·노임정보로 직접 계산, 일위대가표는 표준화된 단위공정의 단가표.

### 단가산출서 양식

단가산출서는 공종별 노무·재료·경비를 직접 산정한 양식이다.

![단가산출서 양식 예시](/procurement-wiki/images/textbook/book4/p128_fig2.png)

> 📌 **실무 포인트**: 양식의 구성(노무량·노임단가·재료량·단가)을 이해하면 직접공사비내역서로의 흐름을 파악할 수 있다.

### 직접공사비 세부공종별 내역서

직접공사비내역서는 공종별로 재료비·노무비·경비를 상세 집계한 결과이다.

![직접공사비 세부공종별 내역서](/procurement-wiki/images/textbook/book4/p132_fig1.jpeg)

> 📌 **실무 포인트**: 작성 순서는 위계구조의 **역순**(단가→단가산출서→직접공사비내역서→공사원가계산서)이라는 점이 중요.

### 공사원가계산서 예시

공사원가계산서는 직접·간접공사비, 일반관리비, 이윤, 부가가치세를 총괄하는 최종 양식이다.

![공사원가계산서 예시](/procurement-wiki/images/textbook/book4/p134_fig1.png)

> 📌 **실무 포인트**: 공사 이윤율 **15%** 적용. 계약이행보증서·하자이행보증수수료는 경비에 포함되지 않는다는 함정 출제 핵심.

## 5. 노무비

- 직접노무비 단가: **대한건설협회** 시중노임단가 적용 (매년 상·하반기 2회 공표)
- 노무량: **표준품셈** 적용
- 간접노무비 = 직접노무비 x 간접노무비율
- 간접노무비는 직접노무비 초과 불가 (기계화·자동화로 불가피한 경우 예외)

## 6. 재료비 특수 사항

| 항목 | 계상 방법 |
|------|-----------|
| 공구손료 | 해당공종 노무비 x 품셈률(3% 이내) |
| 소모잡자재 | 주재료비 x 품셈률(2~5%) |
| 작업설·부산물 | 재료비에서 공제 (기존 시설물 철거 시 제외) |
| 외주가공품(완성품) | 재료비로 계상 |
| 외주가공비(실가공비) | 경비로 계상 |

## 7. 검사(검수) 절차

공사계약은 물품·용역과 달리 **기성청구**(공정별 완성부분 대가)와 **준공확인**(최종 완공) 절차가 구분된다.

## 핵심 암기 포인트

- 공사 이윤율: **15%**
- 공사 노임단가: **대한건설협회** (매년 상·하반기 2회)
- 공사원가계산 순서: 단가 → 단가산출서/일위대가표 → 직공비내역서 → 공사원가계산서
- 작업설·부산물: 재료비에서 **공제** (기존 시설물 철거 시 **공제하지 않음**)
- 국가계약법 공사: 매각비용 별도 계상 또는 분리발주
- 지방계약법 공사: **분리발주 원칙**

## OX 퀴즈

**Q1.** 공사원가계산서는 단가산출서보다 먼저 작성한다.
→ **X** (작성 순서는 단가산정 → 단가산출서/일위대가표 → 직공비내역서 → 공사원가계산서이다)

**Q2.** 공사원가계산에서 간접노무비는 어떤 경우에도 직접노무비를 초과할 수 없다.
→ **X** (원칙적으로 초과 불가하나, 공사현장의 기계화·자동화로 불가피한 경우 증빙자료에 따라 초과 계상 가능)

**Q3.** 기존 시설물 철거 시 발생하는 작업설과 부산물은 재료비에서 공제한다.
→ **X** (기존 시설물의 철거·해체·이설로 발생하는 작업설·부산물은 재료비에서 공제하지 않는다)

**Q4.** 계약이행보증서 수수료는 공사원가의 경비에 포함된다.
→ **X** (계약이행보증서와 하자이행보증수수료는 경비에 포함되지 않는다)

## 함정 노트

⚠️ 재료 구입 **과정**에서 발생하는 부대비용 → 재료비에 계상 / 재료 구입 **후** 발생하는 부대비용 → 경비로 계상

⚠️ 지방계약법 대상 공사의 작업설·부산물은 **분리발주가 원칙**
