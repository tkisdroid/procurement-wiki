# 제5장 전략적 조달의 활용

## 개요

전략적 공공조달의 구체적 실행 제도인 **우선구매(Set-aside)** 제도를 중심으로, 중소기업·녹색·혁신·사회적 책임 조달의 실무적 적용을 다루는 장이다.

|---|--------|----------|
| 제1절 | [[전략적조달의활용_우선구매제도]] | 별도배정(Set-aside) 개요, 미국·캐나다·호주 사례 |
| 제2절 | [[전략적조달의활용_중소기업지원]] | 중소기업자간 경쟁, 고시금액, 지역제한 |
| 제3절 | [[전략적조달의활용_녹색조달]] | 녹색제품 우선구매, 환경표지 |
| 제4절 | [[전략적조달의활용_혁신조달]] | 기술개발제품, 우수제품, 혁신제품 |

## 우선구매제도 국제 비교

| 국가 | 명칭 | 대상 | 목표 |
|------|------|------|------|
| **미국** | Set-aside | 소기업, 여성, 장애 재향군인, HUBZone | 연방계약의 23% (소기업) |
| **캐나다** | PSIB | 원주민기업 | 총 계약금액의 5% |
| **호주** | IPP | 원주민기업 | 물량·가치 양 기준 연간 목표 |
| **한국** | 우선구매 | 중소기업, 여성, 장애인, 사회적기업 | 50% 이상 (중소기업) |

## 한국 우선구매 핵심 수치

| 항목 | 기준 |
|------|------|
| 중소기업 구매목표 | 물품·용역 공급실적의 **50% 이상** |
| 여성기업 | 물품·용역 **5%** 이상, 공사 **3%** 이상 |
| 장애인기업 | **1%** 이상 |
| 중소기업 우선조달 | 고시금액(**2.3억 원**) 미만 |
| 소기업·소상공인 우선 | **1억 원** 미만 |
| 지역제한(물품·용역) | 추정가격 **3.3억 원** 미만 (일부 5.0억) |
| 지역제한(종합공사) | 추정가격 **100억 원** 미만 |
| 지역제한(전문공사) | 추정가격 **10억 원** 미만 |

## 미국 Set-aside 구체적 목표

| 대상 | 목표비율 |
|------|---------|
| 소기업 전체 | **23%** |
| 여성 소유 소기업 | **5%** |
| 소기업(일반) | **5%** |
| HUBZone 소기업 | **3%** |
| 서비스 장애 재향군인 소유 | **3%** |
| 15만 달러 미만 계약 | 자동 소기업 배정 |

---

## 핵심 암기 포인트

- 한국 중소기업 구매목표: **50% 이상** / 여성 **5%/3%** / 장애인 **1%**
- 고시금액: **2.3억 원** (중소기업 우선조달 기준)
- 미국 소기업 목표: 연방계약의 **23%**
- 캐나다 PSIB 원주민 목표: **5%**
- 미국 15만 달러 미만 → **자동** 소기업 배정
- 지역제한: 물품·용역 **3.3억** 미만, 종합공사 **100억** 미만, 전문공사 **10억** 미만

## OX 퀴즈

**Q1.** 한국의 중소기업 공공조달 구매목표비율은 30% 이상이다.
→ **X** (50% 이상이다)

**Q2.** 미국에서 15만 달러 미만의 정부 계약은 자동으로 소기업에 배정된다.
→ **O**

**Q3.** 캐나다의 PSIB(원주민조달전략)의 목표는 총 계약금액의 10%이다.
→ **X** (5%이다)

**Q4.** 한국에서 추정가격 2.3억 원 미만의 조달은 중소기업에 우선 조달 기회가 부여된다.
→ **O** (고시금액 미만)

**Q5.** EU는 미국처럼 구체적인 중소기업 공급 물량 할당 제도를 적극적으로 도입하고 있다.
→ **X** (EU는 할당을 경쟁 제한 차별조치로 간주하여 적극 도입하지 않았으나, 최근 일부 국가에서 시도 중)

## 함정 노트

⚠️ **고시금액 2.3억 원**: 중소기업 우선조달의 기준금액 — 시험 빈출

⚠️ **여성기업 목표 이중 기준**: 물품·용역은 5%, 공사는 3% — 구분 필요

⚠️ **Set-aside ≠ 독점**: 우선 기회 부여이지 독점이 아님, 경쟁 원칙은 유지
