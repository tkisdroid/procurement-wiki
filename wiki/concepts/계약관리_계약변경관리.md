# 제2절 계약변경 관리

## 2.1 변경계약 관리 일반절차

계약은 확정 내용대로 이행이 원칙이나, 「국가계약법」 제19조는 **물가변동·설계변경·기타 계약내용 변경**으로 계약금액 조정을 인정한다.

### 변경계약 처리 절차
1. 변경계약 가능 여부 판단
2. 합의각서 작성 (발주담당자-계약상대자)
3. 변경계약 의뢰 (방침서, 합의각서, 산출내역서 등)
4. 변경계약서 초안 송신 (나라장터)
5. 보증서·인지세 등 제출, 응답서 제출
6. 변경계약 체결 (dBrain 등록)

## 2.2 물가변동에 따른 계약금액 조정 (Escalation / De-Escalation)

### 가. 조정요건 (동시 충족 필수)

| 요건 | 내용 |
|------|------|
| 기간 요건 | 계약 체결일(장기계속은 1차 체결일)로부터 **90일** 이상 경과 |
| 등락 요건 | 품목조정률 또는 지수조정률이 **3%** 이상 증감 |
| 조정기준일 | 두 요건이 **동시에 충족**되는 최초의 날 |
| 재조정 제한 | 조정기준일로부터 **90일** 이내 재조정 불가 |

### 나. 조정 방법 (택일, 변경 불가)

| 구분 | 품목조정률 | 지수조정률 |
|------|-----------|-----------|
| 방식 | 개별 비목 등락률 계산 | 비목군 편성 후 지수 적용 |
| 적용대상 | 거래실례가격·원가계산 예정가격 계약 | 원가계산 예정가격 계약 |
| 장점 | 실제 물가변동 반영 | 조정률 산출 용이 |
| 단점 | 계산 복잡 | 실제 반영 부정확 가능 |
| 적합 | 단기·소규모·단순공종 | 장기·대규모·복합공종 |

- 계약 체결 시 계약서에 명시 (상대자가 지수조정률 원하지 않으면 품목조정률)
- **계약 이행 도중 변경 불가**

### 다. 조정신청

- **증액**: 계약상대자 신청 필요 (자동 조정 아님, 대법원 2006.9.14. 2004다28825)
- **감액**: 발주기관이 최종 대가 지급 전까지 감액내역 통보
- 신청 시기: **대가 수령 전**까지
- 조정 처리: 청구받은 날부터 **30일** 이내

### 라. 조정금액 산출

**증감액 = 물가변동적용대가 x 조정률**

- **물가변동적용대가**: 조정기준일 이후 이행부분 대가
- 공정예정표상 조정기준일 이전 완료 예정 → 지체된 경우 조정 대상 **제외** (단, 정부 귀책·불가항력 시 포함)
- **선금공제**: 조정기준일 **이전** 지급분만 공제 (이후 지급분·당일 지급분은 공제 안 함)
- **감액조정 시 선금공제 하지 않음**

### 마. 특례규정

| 구분 | 내용 |
|------|------|
| 기간요건 완화 | 천재지변·원자재 급등 시 90일 이내 조정 가능 |
| 완화 기준 | 공사·용역·물품제조: 조정률 **5%↑**, 물품구매: **10%↑** |
| 단품 슬라이딩 | 특정 자재 가격증감률 **15%↑** 시 해당 자재만 조정 (공사계약) |

## 2.3 설계변경에 따른 계약금액 조정

### 설계변경 사유
1. 설계서 불분명·누락·오류·상호 모순
2. 지질·용수 등 현장 상태가 설계서와 상이
3. 새로운 기술·공법으로 공사비 절감·공기 단축 효과 현저
4. 기타 발주기관이 필요하다고 인정

### 설계변경 단가 비교 (일반공사)

| 구분 | 계약상대자 요구 | 발주기관 요구 | 신기술·신공법 |
|------|----------------|--------------|--------------|
| 감소물량 | 계약단가 | 계약단가 | 계약단가 |
| 증가물량 | 계약단가 (예가>계약시 예가) | 협의(불가시 50%) | 계약단가(예가>계약시 예가) |
| 신규비목 | 설계변경당시 단가 x 낙찰률 | 협의(불가시 50%) | 설계변경당시 단가 x 낙찰률 |

- 신기술·신공법 적용 시: 절감액의 **30%만 감액**, 70%는 수급인 이익
- 턴키(설계·시공 일괄) 계약: 정부 귀책·불가항력 외 **증액 불가**

### 설계변경 제한
- 일괄입찰, 대안입찰(대안 채택 부분), 기술제안입찰(채택 부분): **정부 귀책·불가항력 외 증액 불가**

## 2.4 기타 계약내용 변경에 따른 조정

- 공사이행기간 변경, 운반거리 변경 등
- 설계변경은 있으나 **공사물량 증감 없음** (설계변경 조정과의 차이)
- **실비를 초과하지 않는 범위**에서 조정

## 2.5~2.6 계약의 해제·해지

| 구분 | 법정 해제·해지 | 합의 해제·해지 |
|------|---------------|---------------|
| 원인 | 채무불이행 등 법정 사유 | 당사자 합의 |
| 정산 | 원상회복 원칙 | 합의 내용에 따름 |
| 손해배상 | 청구 가능 | 특약 없으면 불가 |

## 2.7 분쟁해결 절차

### 민사상 분쟁해결
- 민사보전·집행 (가압류, 가처분)
- 민사소송 (이행·확인·형성청구)
- ADR: 화해, 조정, 중재

### 행정상 분쟁해결
- 행정심판 (부정당업자 제재 등)
- 행정소송 (항고소송, 취소소송)
- 집행정지

**Q1.** 물가변동 조정은 계약 체결일부터 60일이 경과하면 가능하다.
→ **X** (90일 이상 경과해야 한다)

**Q2.** 품목조정률과 지수조정률은 같은 계약에서 동시에 적용할 수 있다.
→ **X** (하나의 방법만 선택, 이행 도중 변경 불가)

**Q3.** 물가변동으로 계약금액이 증액되는 경우 계약상대자의 신청 없이 자동 조정된다.
→ **X** (계약상대자의 적법한 신청에 의해 이루어진다 — 대법원 판례)

**Q4.** 조정기준일 당일 수령한 선금은 물가변동 조정 시 공제 대상이다.
→ **X** (조정기준일 이후 및 당일 수령 선금은 공제하지 않는다)

**Q5.** 단품 슬라이딩은 특정 자재 가격이 10% 이상 변동 시 적용된다.
→ **X** (15% 이상이다)

**Q6.** 설계·시공 일괄입찰(턴키) 계약에서도 발주기관 귀책사유 시 계약금액 증액이 가능하다.
→ **O**

**Q7.** 설계변경 시 신기술·신공법 적용으로 절감된 공사비 중 100%가 수급인 이익이다.
→ **X** (절감액의 30% 감액, 70%가 수급인 이익)

**Q8.** 기타 계약내용 변경에 따른 조정은 실비를 초과할 수 없다.
→ **O**

**Q9.** 물가변동 감액조정 시에도 선금을 공제한다.
→ **X** (감액조정 시에는 선금공제를 하지 않는다)

**Q10.** 중재판정은 양측 당사자에게 확정판결과 같은 효력이 있다.
→ **O**

## 함정 노트

⚠️ 물가변동 "조정기준일" ≠ 조정신청일 ≠ 실제 조정일. 조정기준일은 **두 요건 동시 충족 최초일**이다.

⚠️ 품목조정률/지수조정률 선택 후 **이행 도중 변경 불가** — 1회차에 지수조정률 적용 시 2회차도 동일.

⚠️ 물가변동적용대가 산정 시 **공정예정표** 기준이며, 계약상대자 귀책 지체분은 제외.

⚠️ 설계변경 단가: "발주기관 요구" 시 협의 → 불가 시 **(설계변경당시 단가 + 동 단가 x 낙찰률) / 2** = 50%

⚠️ 단품 슬라이딩: 총액 조정요건(3%) 충족 시에는 총액 조정 우선 → 단품은 총액 미충족 시에만 적용.
