# 제3장 공급업체의 공급계획 수립

## 개요

공급업체가 공공조달에 체계적으로 참여하기 위한 **5단계 공급계획 수립 절차**이다.

| 단계 | 내용 |
|:----:|------|
| 1. 수요정보 수집 | 공공부문의 공급 가능 물품·서비스·공사 수요 수준 확인 |
| 2. 공급환경분석 (법적 적정성) | 법령, 정책, 제도 분석 |
| 3. 내부공급역량 분석 (기술적 적정성) | 공공부문 수요 충족도/적합도 |
| 4. 적정성 분석 (종합) | 법적, 기술적, 경제적 수용 적정성 |
| 5. 공급계획수립 | 입찰서/제안서/견적서 제출 준비 및 계획 |

## 수요정보 수집 방법

- **나라장터 발주목록**: 분기별 발주계획 공개 (최대 3개월 전 확인 가능)
- **지방자치단체 홈페이지**: 자체 발주 및 중앙조달 요청 사업 정보
- **조달청 조달데이터허브**: 중앙조달/자체조달 수요 특성, 계약방법별 분석, 품명별 분석, 입찰혼잡도 분석

## 조달·입찰요청·계약 방법 비교

| 일반적 조달방법 | 입찰요청 방법 | 국내 계약방법 |
|----------------|--------------|--------------|
| 입찰(경쟁) | RFQ | 제한적 최저가, 희망수량경쟁 |
| 입찰(경쟁) | ITB | 적격심사, 2단계경쟁, 종합낙찰제 |
| 입찰(경쟁) | RFP | 협상에 의한 계약, 경쟁적 대화 |
| 수의계약 | RFQ | 제한적 최저가 |
| 소액견적수의 | RFQ (2인 이상) | 물품 1억, 종합공사 4억, 전문공사 2억 이하 |
| 1인견적수의 | RFQ (1인) | **2천만원 이하** |
| 종합쇼핑몰 | 납품요구/2단계경쟁 | MAS, 제3자단가계약 |
| 구매카드 | 소액구매 | 국가기관 **1백만원**, 지자체 **3백만원** (건당) |

## 원가계산 방법 (물품제조)

### 제조원가 구성

| 비목 | 세부 | 내용 |
|------|------|------|
| **재료비** | 직접재료비 | 주요재료비 + 부분품비 |
| | 간접재료비 | 소모재료비 + 소모공구비 + 포장재료비 |
| **노무비** | 직접노무비 | 기본급 + 제수당 + 상여금(연400%한도) + 퇴직급여충당금 |
| | 간접노무비 | 보조작업 노무자, 현장감독자 급여 |
| **경비** | - | 재료비·노무비 제외 제조원가 |
| **일반관리비** | - | 제조원가에 업종별 비율 적용 |
| **이윤** | - | (노무비+경비+일반관리비)의 **25% 이하** (기술료·외주가공비 제외) |

## 핵심 암기 포인트

- 공급계획 수립: **5단계** (수요정보수집 → 법적 → 기술적 → 종합 → 공급계획)
- 나라장터 발주계획: **분기별** 공개, 최대 **3개월 전** 확인 가능
- 1인견적수의: **2천만원 이하**
- 소액구매(구매카드): 국가기관 **1백만원**, 지자체 **3백만원** (건당)
- 이윤 한도: **(노무비+경비+일반관리비)의 25% 이하**
- 상여금 한도: 기본급 기준 연 **400%**

## OX 퀴즈

**Q1.** 나라장터 발주계획 정보를 통해 최대 6개월 전에 공급 기회를 파악할 수 있다. → **X** (분기별 공개로 최대 3개월 전)

**Q2.** 1인견적수의계약은 3천만원 이하인 경우에 활용된다. → **X** (2천만원 이하)

**Q3.** 물품제조 원가계산에서 이윤은 노무비, 경비, 일반관리비 합계의 25%를 초과할 수 없다. → **O**

**Q4.** 다수공급자계약(MAS) 물품은 일반 입찰공고를 통해서도 공급할 수 있다. → **X** (MAS 체결 후 종합쇼핑몰에 등록해야 공급 가능)

**Q5.** 구매카드를 통한 소액구매는 국가기관과 지자체의 건당 한도가 동일하다. → **X** (국가기관 1백만원, 지자체 3백만원)

## 함정 노트

- ⚠️ 발주계획 확인의 중요성: 입찰공고에서 정보를 인지하면 이미 **최적의 준비 기간을 놓친 것**
- ⚠️ 물품제조 vs 물품구매: 동일 품명이라도 **제조** 공고 시 제조업체만 참여 가능, **구매** 공고 시 모든 업체 참여 가능
- ⚠️ 법적 적정성 검토의 중요성: 입찰공고의 법적 문제점 발견 시 **사전규격공개 또는 입찰 마감 전에 의견등록**하여 개선 요청
