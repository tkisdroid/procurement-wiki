# 제1절 용역계약 절차 및 이행관리

## 1.1 관련 법규 체계
- 「국가계약법」·「지방계약법」
- 「용역계약일반조건」(계약예규)
- 「일반용역계약특수조건」(조달청)

## 1.2 용역계약 종류

### 계약금액 산정 여부별

| 종류 | 설명 |
|------|------|
| 총액계약 | 전체 총액으로 체결 |
| 일반단가계약 | 단가로 체결, 반복 제조·공급 |
| 제3자를 위한 단가계약 | 수요기관이 직접 납품 요구 |
| MAS (다수공급자계약) | 2인 이상 계약상대자, 수요기관 선택 |
| 카탈로그계약 | 규격화 곤란 용역, 카탈로그 기반 |

### 계약금액 확정 여부별

| 종류 | 설명 |
|------|------|
| 확정계약 | 금액 확정 (통상) |
| 개산계약 | 미리 정할 수 없을 때 (시험·조사·연구, 긴급 재해복구) |
| 사후원가검토 조건부 | 일부 비목 금액 미결정 |

### 계약기간별
- 단년도 계약, 장기계속계약, 계속비계약

## 1.3 용역계약 입찰 방식

| 방식 | 설명 |
|------|------|
| 일반경쟁 | 불특정 다수 참여, 기본원칙 |
| 제한경쟁 | 실적·기술·지역 등 제한 |
| 지명경쟁 | 특정 다수 지명, 10인 이내 |
| 수의계약 | 특정 상대, 예외적 인정 |

### 제한경쟁 실적제한 기준 (고시금액 이상)
- 규모·양 기준: 해당 목적물의 **1배** 이내
- 금액 기준: 추정가격의 **1배** 이내
- 지역제한: 고시금액 미만(2.3억) → 법인등기부 본점 소재지

---

## OX 퀴즈

**Q1.** 개산계약은 계약금액을 미리 확정하여 체결하는 계약이다.
→ **X** (개산가격으로 체결하고 확정 후 정산하는 계약)

**Q2.** 지명경쟁은 입찰대상자가 20인 이내인 경우 가능하다.
→ **X** (10인 이내이다)

**Q3.** 일반경쟁입찰이 용역계약 입찰방식의 기본원칙이다.
→ **O**

**Q4.** 제한경쟁 시 실적제한은 추정가격의 2배 이내이다.
→ **X** (1배 이내이다)

---

## 함정 노트

⚠️ 지명경쟁 **10인** 이내 — "20인"으로 바꿔 출제 빈출.

⚠️ 개산계약 적용 대상: 시제품 제조, 시험·조사·연구 용역, 긴급 재해복구 등.
