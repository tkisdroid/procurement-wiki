# 제1절 용역계약 절차 및 이행관리

## 1.1 관련 법규 체계
- 「국가계약법」·「지방계약법」
- 「용역계약일반조건」(계약예규)
- 「일반용역계약특수조건」(조달청)

## 1.2 용역계약 종류

### 계약금액 산정 여부별

| 종류 | 설명 |
|------|------|
| 총액계약 | 전체 총액으로 체결 |
| 일반단가계약 | 단가로 체결, 반복 제조·공급 |
| 제3자를 위한 단가계약 | 수요기관이 직접 납품 요구 |
| MAS (다수공급자계약) | 2인 이상 계약상대자, 수요기관 선택 |
| 카탈로그계약 | 규격화 곤란 용역, 카탈로그 기반 |

### 계약금액 확정 여부별

| 종류 | 설명 |
|------|------|
| 확정계약 | 금액 확정 (통상) |
| 개산계약 | 미리 정할 수 없을 때 (시험·조사·연구, 긴급 재해복구) |
| 사후원가검토 조건부 | 일부 비목 금액 미결정 |

### 계약기간별
- 단년도 계약, 장기계속계약, 계속비계약

## 1.3 용역계약 입찰 방식

| 방식 | 설명 |
|------|------|
| 일반경쟁 | 불특정 다수 참여, 기본원칙 |
| 제한경쟁 | 실적·기술·지역 등 제한 |
| 지명경쟁 | 특정 다수 지명, 10인 이내 |
| 수의계약 | 특정 상대, 예외적 인정 |

### 제한경쟁 실적제한 기준 (고시금액 이상)
- 규모·양 기준: 해당 목적물의 **1배** 이내
- 금액 기준: 추정가격의 **1배** 이내
- 지역제한: 고시금액 미만(2.3억) → 법인등기부 본점 소재지

## 1.4 일반용역 계약 절차

![일반용역 계약 절차](/procurement-wiki/images/textbook/book3/p176_fig1.jpeg)

> 📌 **학습 포인트**: 수요발생 → 조달요청 → 사전규격공개(5~10일) → 구매결의 → 입찰공고 → 개찰 → 낙찰자 선정 → 계약 체결까지 일반용역 총액계약(중앙조달)의 전체 흐름을 한 장으로 보여준다. "사전규격공개 5일(소프트웨어 5억 이상은 10일)" 같은 일정 숫자가 핵심.

## 1.5 일반용역 낙찰자 결정 방법

### 1) 적격심사

![일반용역 적격심사 절차](/procurement-wiki/images/textbook/book3/p180_fig1.jpeg)

> 📌 **학습 포인트**: 적격심사 통과점수는 **85점**(소프트웨어 중기간 경쟁제품·육상운송용역은 88점). 5억 원 미만/이상에 따라 입찰가격 비중이 70/60으로 달라지는 점을 그림과 함께 외우면 변형 출제에 강해진다.

### 2) 2단계 경쟁 및 규격·가격 동시입찰

![2단계 경쟁 및 규격·가격 동시입찰 절차](/procurement-wiki/images/textbook/book3/p182_fig1.jpeg)

> 📌 **학습 포인트**: 2단계 경쟁은 규격(기술)입찰과 가격입찰이 **각각 2인 이상** 유효해야 성립하지만, 동시입찰(Two Envelope)은 하나의 입찰로 보아 기술적격자 1인이어도 가격개찰이 가능하다는 차이가 핵심.

### 3) 협상에 의한 계약

![협상에 의한 계약 절차](/procurement-wiki/images/textbook/book3/p183_fig1.jpeg)

> 📌 **학습 포인트**: 제안서 제출 → 기술평가 → 협상대상자 선정 → 협상(15일 이내, 10일 연장 가능) → 낙찰자 결정의 흐름. 기술능력 평가 70~90점 vs 입찰가격 평가 10~30점 비중과 함께 묻는다.

![사업수행능력평가 근거 법령 평가체계](/procurement-wiki/images/textbook/book3/p184_fig1.png)

> 📌 **학습 포인트**: 협상에 의한 계약 평가의 근거 법령(국가계약법·지방계약법·정보화사업 지침)별 기술능력/입찰가격 배점이 다르다는 점을 정리한 표이다. 기재부 기준 70:30, 지방 기준 80:20처럼 배점 차이가 핵심.

![협상에 의한 계약 제안서 평가 배점표](/procurement-wiki/images/textbook/book3/p184_fig2.png)

> 📌 **학습 포인트**: 기술·지식능력, 사업수행계획, 인력·조직, 수행실적, 재무구조 등 세부 평가항목별 배점을 정리한 표. 협상계약은 "최저가"가 아니라 "기술+가격 종합점수 최고자"임을 시각적으로 보여준다.

### 4) 경쟁적 대화에 의한 계약

![경쟁적 대화 방식에 의한 계약 절차](/procurement-wiki/images/textbook/book3/p185_fig1.jpeg)

> 📌 **학습 포인트**: 기본 제안서 제출 → 경쟁적 대화(2회 이상) → 최종제안서 → 평가·낙찰. 입찰공고는 제안서 마감 **40일 전**(긴급·고시금액 미만·재공고는 10일)이라는 일정 숫자가 시험에 잘 나온다.

![경쟁적 대화 방식 제안서 평가항목 및 배점한도](/procurement-wiki/images/textbook/book3/p186_fig1.png)

> 📌 **학습 포인트**: 경쟁적 대화는 기술능력 **90점** + 입찰가격 **10점**으로, 협상계약(70:30)보다 기술 비중이 훨씬 크다. 기술능력 세부항목 배점한도 50점 / 지원기술·사후관리 10점 한도도 함께 외워야 한다.

## 1.6 기술용역 낙찰자 결정 방법

### 1) PQ + 적격심사

![기술용역 적격심사 절차](/procurement-wiki/images/textbook/book3/p188_fig1.png)

> 📌 **학습 포인트**: 기술용역은 **PQ(사업수행능력평가) → 적격심사** 순서로 진행되며, 「건설기술진흥법」상 고시금액 이상의 설계·건설사업관리 용역에 적용된다. 일반용역의 적격심사와 평가 기준이 다르다는 점이 출제 포인트.

### 2) 종합심사낙찰제

![종합심사낙찰제 절차](/procurement-wiki/images/textbook/book3/p189_fig1.png)

> 📌 **학습 포인트**: 기술용역 종합심사 적용 대상은 추정가격 ①건설사업관리 50억 이상 ②기본계획·기본설계 30억 이상 ③실시설계 40억 이상으로, 금액 기준이 시험에 자주 나온다.

### 3) 설계공모

![설계공모 절차](/procurement-wiki/images/textbook/book3/p191_fig1.jpeg)

> 📌 **학습 포인트**: 설계공모는 추정가격 **1억 원 이상** 건축설계용역에 우선 적용. 일반설계공모(공모기간 90일 이상) / 2단계설계공모(1단계 30일+2단계 60일) / 제안공모(20일 이상)의 3가지 방식 차이를 묻는다.

## OX 퀴즈

**Q1.** 개산계약은 계약금액을 미리 확정하여 체결하는 계약이다.
→ **X** (개산가격으로 체결하고 확정 후 정산하는 계약)

**Q2.** 지명경쟁은 입찰대상자가 20인 이내인 경우 가능하다.
→ **X** (10인 이내이다)

**Q3.** 일반경쟁입찰이 용역계약 입찰방식의 기본원칙이다.
→ **O**

**Q4.** 제한경쟁 시 실적제한은 추정가격의 2배 이내이다.
→ **X** (1배 이내이다)

## 함정 노트

⚠️ 지명경쟁 **10인** 이내 — "20인"으로 바꿔 출제 핵심.

⚠️ 개산계약 적용 대상: 시제품 제조, 시험·조사·연구 용역, 긴급 재해복구 등.
