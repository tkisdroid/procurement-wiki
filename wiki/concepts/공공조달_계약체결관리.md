# 제5장 공공조달 계약 체결 관리

## 하위 절

| 절 | 제목 | 핵심 내용 |
|---|------|----------|
| 제1절 | 계약체결 | 전자계약 절차, 변경계약, 해지·해제 |
| 제2절 | 착수보고와 착공(착수)계 제출 | 착공계·착수계 등록, 선금 청구 |
| 제3절 | 계약이행 보증 관리 | 보증서, 네트워크론 |

## 1. 전자계약 체결 절차

```
발주기관: 계약서 초안 송신 → 조달업체: 계약응답서 회신
→ 발주기관: 계약체결통보서 송신 → 계약 체결 완료
```

- 계약응답서 작성 시 **지급각서 및 청렴서약서** 동의 필수
- 계약서 초안에 이의가 있으면 **반려 사유** 첨부하여 회신 가능
- 모든 전자계약 절차는 **인증서 전자서명** 필수

## 2. 변경계약

- 변경 사유: 공공계약법령·계약예규에서 규정한 사유 (내용, 예산, 과업기간 등)
- 절차: 조달업체 변경요청서 → 발주기관 승인 → 변경계약서 초안 → 응답서 → 통보서
- 당초 계약기간 초과 시 **지체상금 부과 또는 면제** 여부 명시
- 정당한 사유로 기간 연장 시 지체상금 면제 가능

## 3. 착공(착수)계

| 구분 | 명칭 |
|------|------|
| 공사 | 착공계 |
| 물품·용역 | 착수계 |

- 계약 체결 후 이행 시작을 보고하는 절차
- 착공(착수)일자: 계약일자와 동일할 수 있으나, 준비 필요 시 **7~15일** 후 설정 가능
- 인증서 **전자서명** 필수

## 4. 선금 청구

- 착공(착수)계 제출 후 선금 지급 청구 가능
- 조달청 중앙조달: **대지급** 절차로 신속 지급 (조달수수료 수입 회전자금 활용)
- 공동수급체: **대표업체**가 구성업체의 선금 청구 요청 사실 확인 필수

## 5. 계약이행 보증

- 「국가계약법」 제12조에 따라 **계약보증금 또는 보증서** 제출 필수
- 보증기관: 보증보험회사, 공제조합 등 (「국가계약법 시행령」 제37조)
- 변경계약으로 계약기간 연장 시 **보증서도 연장 제출** 필요
- 전자보증요청: 나라장터에서 보증기관에 전자 방식으로 요청 가능

## 6. 네트워크론

- 중소기업 대상, 나라장터 체결 **계약서를 담보**로 저금리 대출
- 별도 담보 없이 계약서만으로 대출 가능
- 대출금 상환: 계약대금 지급 시점에 **자동 상환**
- 절차: 신청서 작성 → 금융기관 심사 → 승인 → 계약체결확인 통보 → 채권양도 통보

## 핵심 암기 포인트

- 전자계약 3단계: **초안 송신 → 응답서 회신 → 통보서 송신**
- 착공(착수)계: 이행 시작 보고, 인증서 전자서명 필수
- 선금 대지급: 조달청이 회전자금으로 신속 지급
- 네트워크론: 중소기업 대상, 계약서 담보, 자동 상환
- 보증서: 변경계약으로 기간 연장 시 **추가 제출** 필요

## OX 퀴즈

**Q1.** 조달업체는 계약서 초안에 이의가 있으면 반려할 수 있다.
→ **O** (반려 사유를 첨부하여 회신할 수 있다)

**Q2.** 네트워크론은 대기업도 이용할 수 있다.
→ **X** (중소기업을 대상으로 한 제도이다)

**Q3.** 공동수급체의 구성업체도 직접 선금을 청구할 수 있다.
→ **X** (대표업체가 선금 청구를 하며, 구성업체의 청구 요청 사실 확인 절차를 진행해야 한다)

**Q4.** 착공(착수)일자는 반드시 계약일자와 동일해야 한다.
→ **X** (준비 기간이 필요한 경우 7~15일 후로 설정 가능하며, 발주기관과 협의하여 결정한다)

**Q5.** 네트워크론 대출금은 조달업체가 수동으로 상환해야 한다.
→ **X** (계약대금 지급 시점에 자동으로 상환이 이루어진다)

## 함정 노트

⚠️ 변경계약으로 기간이 연장되면 **보증서도 해당 기간만큼 연장 제출**해야 한다

⚠️ 공동수급체의 대표업체가 선금을 받고 구성업체에 미지급하는 문제 방지를 위해 **확인 절차** 필수

⚠️ 네트워크론 채권양도통보: 대출받은 조달업체가 **채권은행에 양도 통보** → 채권은행이 조달청에 통보 → 대금 지급 시 대출금 상환
