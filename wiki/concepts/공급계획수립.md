# 제2장 공급계획 수립

## 개요

나라장터 **조달데이터허브**(data.g2b.go.kr)를 활용하여 공급정보를 수집·분석하고, 수요를 예측하며, 공급대상 수요기관을 선정하는 과정이다.

## 하위 절

| 절 | 제목 | 핵심 내용 |
|---|------|----------|
| 제1절 | 공급정보 수집과 분석 | 조달데이터허브, 보고서·오픈API·시각화 활용 |
| 제2절 | 조달데이터 분석 및 예측 | 쇼핑몰키워드 분석, 수요예측, 입찰혼잡도 예측 |
| 제3절 | 공급대상 수요기관 선정 | 발주계획 검색, 사전규격공개, 입찰공고 활용 |

## 1. 조달데이터허브 주요 서비스

| 메뉴 | 기능 | 로그인 필요 |
|------|------|-------------|
| 데이터 제공 | 사전 정의된 보고서 120건+ 다운로드 | 불필요 (추가 보고서는 필요) |
| 오픈 API | 표준화된 조달데이터 웹서비스/앱 개발용 | 불필요 |
| 데이터 시각화 | 그래프 기반 분포·현황 분석 | 불필요 |
| 데이터 분석 | 쇼핑몰키워드분석, 수요예측, 입찰혼잡도 예측 | 일부 필요 |

**사용자 유형별 차별적 보고서**: 조달청, 수요기관, 조달업체, 일반 국민 (4개 유형)

## 2. 주요 분석 기능

### 쇼핑몰키워드 분석
- 세부품명별 키워드 검색 빈도(검색량)와 기관별 납품 실적 확인
- 대안 키워드 사용 빈도 확인 → 상품설명 키워드 설정에 참고
- 수요기관 유형별 납품 실적 현황 분석
- 급상승 상품 상위 10개 확인 가능

### 공공조달 수요예측
- 중앙조달 vs 자체조달 구분하여 수요 규모 변화 추세 제공
- 물품·용역·공사별 공급 실적 기반 향후 전망
- 광역자치단체별, 계약 방법별, 물품분류별 예측 분석

### 입찰혼잡도 예측
- 주간 단위 기준 일자별·시간대별 혼잡도 **6단계**로 세분화
- 입찰 집중 시간 회피로 네트워크 과부하·시스템 접속 실패 예방

### 빅데이터 분석
- 물품 이미지 정보를 활용한 적합 세부품명 추천 기능

## 3. 수요기관 선정 전략

### 발주계획 활용
- 나라장터 '발주' → '발주계획'에서 당해 연도 발주 예정 사업 확인
- **전년도에 예산 편성 → 차년도 집행** 구조이므로 1년 전 정보 확보 가능
- 발주계획 상세: 발주기관, 발주시기, 조달방식, 계약방법, 예산액 확인

### 입찰 준비 타임라인

| 단계 | 시기 | 준비 기간 |
|------|------|-----------|
| 발주계획 공개 | 당해 연초 | 수개월~1년 전 |
| 사전규격공개 | 입찰공고 전 | 5~10일 여유 |
| 의견 제시 기간 | 사전규격공개 후 5일 | 보완 시 2주 추가 가능 |
| 입찰공고 | 사전규격공개 후 약 7일~ | 5~10일 여유 |

### 지역제한 입찰
- 일반용역 추정가격 **5억 원 미만**: 지역제한 입찰
- 물품 추정가격 기준에 따라 지역제한 적용

## 핵심 암기 포인트

- 조달데이터허브 주소: **data.g2b.go.kr**
- 입찰혼잡도 예측: **주간 단위, 6단계**
- 발주계획: 전년도 예산 편성 → 차년도 집행이므로 **사전 정보 확보 가능**
- 사전규격공개 의견 제시 기간: **5일**
- 통합검색 서비스: **'검색ON'** 기능
- 예정가격 결정: 기초금액 ±3% 범위 내 복수 예비가격 15건 산정
- 적격심사낙찰제: 낙찰하한율 이상 투찰 → 적격심사 대상자 선정

## OX 퀴즈

**Q1.** 조달데이터허브의 보고서와 오픈 API는 로그인하지 않아도 활용 가능하다.
→ **O** (기본 보고서와 오픈 API는 로그인 없이 다운로드·활용 가능하다)

**Q2.** 공공조달 수요예측 기능에서 제공하는 정보는 100% 정확한 예측 결과이다.
→ **X** (빅데이터 활용 예측으로 정확하지 않을 수 있으므로 데이터 보고서와 교차 검증하여 활용하는 것이 효과적이다)

**Q3.** 입찰혼잡도 예측은 월간 단위로 제공된다.
→ **X** (주간 단위를 기준으로 일자별, 시간대별 혼잡도를 제공한다)

**Q4.** 입찰공고 시작 이후 입찰서 또는 제안서 제출 시간적 여유는 약 5~10일이다.
→ **O** (입찰공고 시작 이후 제출 여유가 5~10일뿐이므로 발주계획 단계에서 사전 준비가 중요하다)

**Q5.** 발주계획은 나라장터에서만 확인할 수 있다.
→ **X** (지방자치단체 및 공공기관의 홈페이지 또는 조달·계약 관련 웹페이지에서도 확인 가능하다)

## 함정 노트

⚠️ 쇼핑몰키워드 분석의 검색량 증가가 반드시 납품 기회 증가를 의미하지는 않으므로 실제 공급 실적과 함께 분석해야 한다

⚠️ 복수 예비가격은 기초금액 기준 ±3% 범위 내에서 산정되므로, 이 범위 밖의 투찰은 유효하지 않다

⚠️ 사전규격공개 단계에서 의견이 제시되면 발주기관 내부 검토에 **최대 2주**가 소요될 수 있으며, 이 기간만큼 입찰공고도 지연된다
