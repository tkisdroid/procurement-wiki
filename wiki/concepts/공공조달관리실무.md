# 공공조달 관리실무 (전자조달기반 수행절차 일반)

공공조달관리사 2차 시험과목인 **공공조달 관리실무**는 전자조달시스템(나라장터)을 기반으로 공공조달의 전 과정을 실무적으로 수행하는 절차를 다룬다.

## 전체 구성 (8개 장)

| 장 | 제목 | 핵심 내용 | 중요도 |
|---|------|----------|--------|
| 제1장 | [[공공조달계획수립_나라장터등록]] | 나라장터 사용자 등록, 품명 등록(물품목록화) | high |
| 제2장 | [[공급계획수립]] | 공급정보 수집·분석, 조달데이터 분석·예측, 수요기관 선정 | high |
| 제3장 | [[발주대상_경제성분석_원가계산]] | 공공조달 원가계산, 제조·용역·공사 원가계산 | high |
| 제4장 | [[입찰및낙찰절차관리]] | 제안서 작성, 제안평가, 기술·가격협상 | high |
| 제5장 | [[공공조달_계약체결관리]] | 계약체결, 착수보고, 계약이행보증 관리 | high |
| 제6장 | [[공급대상물_유형별_계약관리]] | 공통절차, 물품·용역·공사 계약 관리 | high |
| 제7장 | [[종합쇼핑몰활용]] | 다수공급자계약(MAS), 우수제품 신청 | high |
| 제8장 | [[공공조달_법제도활용]] | 공공조달법규, 우선구매제도 활용 | medium |

## 핵심 업무 흐름

```
입찰참가자격 등록 → 품명 등록(목록화) → 공급정보 수집·분석
→ 원가계산 → 입찰(제안서 작성/투찰) → 낙찰자 선정
→ 계약 체결 → 착수계 제출 → 계약 이행 → 검사(검수)
→ 대금 청구·지급 → 계약 종결
```

## 핵심 암기 포인트

- 나라장터(www.g2b.go.kr)는 국가종합전자조달시스템으로 모든 공공조달 입찰참가의 기본 자격요건
- 입찰참가자격 등록 후 20일 이내 서류 미제출 시 신청 반려
- 물품분류체계: 대분류(2자리) → 중분류(4자리) → 소분류(6자리) → 세분류(8자리)
- 물품목록번호: 품명번호(8자리) + 식별번호(8자리) = 16자리
- 예정가격 결정 순서: 거래실례가격 → 원가계산 → 표준시장단가 → 감정·유사거래실례·견적가격
- 이윤율: 제조 25%, 공사 15%, 용역 10%
- MAS 입찰공고 기간: 기본 10년, 계약기간: 기본 3년
- 우수제품 지정: 1차 심사(기술수준) → 현장실태조사 → 2차 심사

## 함정 노트

⚠️ 입찰참가자격 등록 정보가 변경되면 즉시 변경 등록해야 하며, 미변경 시 자격 상실 또는 입찰 무효화 가능

⚠️ 제조물품과 공급물품은 구분 등록해야 한다 -- 공급물품은 타사 제조품을 구매·공급하는 경우, 제조물품은 직접 생산하는 경우

⚠️ 원가계산에서 간접노무비는 직접노무비를 초과하여 계상할 수 없음 (공사의 경우 기계화·자동화로 불가피한 경우 예외)

⚠️ 가격협상은 기술(규격)협상에서 과업내용이 가감된 경우에만 진행됨 -- 과업 조정이 없으면 가격협상 불필요
