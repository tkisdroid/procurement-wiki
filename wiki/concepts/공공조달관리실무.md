# 공공조달 관리실무 (전자조달기반 수행절차 일반)

공공조달관리사 2차 시험과목인 **공공조달 관리실무**는 전자조달시스템(나라장터)을 기반으로 공공조달의 전 과정을 실무적으로 수행하는 절차를 다룬다.

## 제1장: 입찰참가자격 등록

모든 공공조달의 출발점은 나라장터 입찰참가자격 등록이다. 등록 없이는 입찰 참여 자체가 불가능하다.

![제1장 입찰참가자격 등록 기능 개요](/procurement-wiki/images/textbook/book4/p8_fig1.png)

> 📌 **실무 포인트**: 등록 절차의 4단계(신청→안내문 인쇄→서류 제출→승인)와 각 단계의 누락 시 처리 결과를 묻는 문제가 중요된다. 자세한 내용은 [[공공조달계획수립_나라장터등록]] 참조.

## 제3장: 원가계산 비목 구성

원가계산은 발주기관의 예정가격과 조달업체의 산출내역서 작성 핵심 도구이다.

![제3장 원가계산 비목 구성도](/procurement-wiki/images/textbook/book4/p99_fig1.jpeg)

> 📌 **실무 포인트**: 재료비·노무비·경비·일반관리비·이윤의 위계와 계산 순서를 외워야 한다. 이윤율(제조 25% / 공사 15% / 용역 10%)은 핵심 포인트.

## 제4장: 협상에 의한 계약 — 제안서 평가

협상에 의한 계약은 가격뿐 아니라 기술 제안 내용을 종합 평가하여 낙찰자를 결정하는 방식이다.

![제4장 협상에 의한 계약 제안서 평가 세부 절차](/procurement-wiki/images/textbook/book4/p149_fig1.png)

> 📌 **실무 포인트**: 제안서 접수 → 평가위원 선정 → 정량·정성평가 → 우선협상대상자 선정 → 기술협상 → 가격협상의 흐름과 각 단계 기간(15일, 10일 등)이 핵심.

## 제6장: 물품계약 업무 흐름

나라장터 물품계약은 입찰공고→투찰→적격심사→낙찰→계약→납품→검수→대금까지 일관된 전자 절차를 따른다.

![제6장 나라장터 물품계약 업무 흐름도](/procurement-wiki/images/textbook/book4/p245_fig1.png)

> 📌 **실무 포인트**: 각 단계에서 조달업체와 발주기관이 수행하는 역할 구분이 출제된다. 복수 예비가격 15개 중 2개 추첨번호 선택은 필수 암기.

## 제6장: 공사계약 절차

공사계약은 PQ심사·종합심사·기성청구·준공확인 등 물품계약과 다른 특화 절차가 있다.

![제6장 공사계약 업무 절차도](/procurement-wiki/images/textbook/book4/p296_fig1.png)

> 📌 **실무 포인트**: PQ심사(사전자격심사) → 입찰 → 적격성심사 또는 종합심사 → 낙찰 → 기성청구 → 준공확인의 흐름과 국가계약법(종합심사) vs 지방계약법(종합평가) 구분 주의.

## 제7장: 다수공급자계약(MAS)

MAS는 조달청이 사전에 2인 이상과 단가계약을 체결하여 수요기관이 종합쇼핑몰에서 즉시 구매하도록 하는 제도이다.

![제7장 다수공급자계약(MAS) 메뉴 화면](/procurement-wiki/images/textbook/book4/p340_fig1.png)

> 📌 **실무 포인트**: 입찰공고 10년·계약기간 3년, 적격성평가(신용등급 B- 이상, 납품실적 3건) 등 숫자 요건 출제 핵심.

## 제8장: 공공조달 법제도 활용

공공조달 법령은 380여 종(자치법규 포함 910여 종)으로 방대해 국가법령정보센터를 통한 최신 법령 확인이 실무 필수이다.

![제8장 국가법령정보센터(법령정보시스템) 접속화면](/procurement-wiki/images/textbook/book4/p395_fig1.png)

> 📌 **실무 포인트**: 법령 검색·해석·판례 활용 방법과 우선구매제도(중소기업/녹색제품/여성·장애인기업) 활용 시스템(공공구매종합정보망, 와우몰, 가치장터, 꿈드래)을 묶어서 학습.

## 핵심 업무 흐름

```
입찰참가자격 등록 → 품명 등록(목록화) → 공급정보 수집·분석
→ 원가계산 → 입찰(제안서 작성/투찰) → 낙찰자 선정
→ 계약 체결 → 착수계 제출 → 계약 이행 → 검사(검수)
→ 대금 청구·지급 → 계약 종결
```

## 핵심 암기 포인트

- 나라장터(www.g2b.go.kr)는 국가종합전자조달시스템으로 모든 공공조달 입찰참가의 기본 자격요건
- 입찰참가자격 등록 후 20일 이내 서류 미제출 시 신청 반려
- 물품분류체계: 대분류(2자리) → 중분류(4자리) → 소분류(6자리) → 세분류(8자리)
- 물품목록번호: 품명번호(8자리) + 식별번호(8자리) = 16자리
- 예정가격 결정 순서: 거래실례가격 → 원가계산 → 표준시장단가 → 감정·유사거래실례·견적가격
- 이윤율: 제조 25%, 공사 15%, 용역 10%
- MAS 입찰공고 기간: 기본 10년, 계약기간: 기본 3년
- 우수제품 지정: 1차 심사(기술수준) → 현장실태조사 → 2차 심사

## 함정 노트

⚠️ 입찰참가자격 등록 정보가 변경되면 즉시 변경 등록해야 하며, 미변경 시 자격 상실 또는 입찰 무효화 가능

⚠️ 제조물품과 공급물품은 구분 등록해야 한다 -- 공급물품은 타사 제조품을 구매·공급하는 경우, 제조물품은 직접 생산하는 경우

⚠️ 원가계산에서 간접노무비는 직접노무비를 초과하여 계상할 수 없음 (공사의 경우 기계화·자동화로 불가피한 경우 예외)

⚠️ 가격협상은 기술(규격)협상에서 과업내용이 가감된 경우에만 진행됨 -- 과업 조정이 없으면 가격협상 불필요
