# 제4절 계약관리

## 4.1 주요 정보

계약관리 단계는 체결된 계약의 이행 과정을 **모니터링하고 추적**하는 절차이다.

### 계약관리단계 핵심 과업

1. **계약관리계획 개발** - 이행 모니터링 절차 수립, 착수회의 진행
2. **계약 및 계약업체 관리** - 성실한 이행과 성과 달성 관리
3. **계약변경관리** - 비용 및 기타 변경 내용 협의·변경계약 체결
4. **계약관리상 문제 및 분쟁 관리** - 분쟁 해결, 중지, 자격정지, 해제/해지
5. **계약목적물 인수** - 검사 절차 후 인수 결정
6. **대금지급 및 지불기록 관리** - 인수 확정 물품의 대금지급

## 4.2 계약관리계획(CMP) 개발

- CMP: Contract Management Plan
- 6하원칙(누가, 무엇을, 언제, 어디서, 어떻게, 왜) 중심으로 수립
- **'투입물 → 프로세스 → 결과물'**의 흐름
- 고위험 또는 매우 복잡한 조달사업에 한정하여 수립

### CMP 고려사항
1. 프로젝트 설명, 수행 기간, 일정, 마일스톤, 납품일
2. 핵심 공정(Critical Path) 과업과 결과물
3. 계약당사자 간 역할과 책임
4. 관리 자료와 보고 방법
5. 검사·인수, 투입 인력, 테스트, 보증 조항
6. 감시 항목, 특별 약관, 보험, 변경, 문제 해결 절차

### 착수회의(킥오프 미팅)
- 계약 이행 공식 시작 알림
- 요구사항과 역할·책임에 대한 공통 이해 형성
- 변경 사항 관리, 문제 식별, 통지·해결 절차 논의

## 4.3 계약과 계약업체 관리

### 계약 모니터링 7단계
1. 계약서와 계약문서 수집
2. 모니터링 대상 결정
3. 모니터링 방법 결정
4. 모니터링 담당자 결정
5. 정보와 데이터 수집
6. 정보와 데이터 분석
7. 계약 불이행 시 조치사항 마련

### 계약업체 관계 관리 지침

| 원칙 | 내용 |
|------|------|
| 권리/의무 명확성 | 계약 내용을 숙지하고 권리와 의무를 명확히 인식 |
| 협의된 기대치 설정 | 기대되는 성과수준을 계약초기에 설정 |
| 공정한 대우 | 신뢰할 수 있는 정보 제공, 윤리적 활동 |
| 합리성 | 합리적인 요구와 절차 수행 |
| 협력성 | 장기적 협력 관계 구축 |
| 의사소통 | 정기적 의사소통 보장 |
| 문서화 | 정보와 의사소통 내용 문서화 |
| 대가지급 | 인수 시 **즉시 대가 지급** |

## 4.4 계약 변경 관리

변경 사유:
1. 과업 내용(구매사양, 설계서, 시방서 등) 변경
2. 일정 수준 초과 비용 변동
3. 과업 범위, 비용, 계약목적물 기능에 큰 영향을 미치는 변경

## 4.5 계약관리상 문제와 분쟁의 관리

### 에스컬레이션 프레임워크 (5단계)

| 단계 | 내용 |
|:----:|------|
| 1 | 계약상대자의 현장 관리자와 문제해결 시도 |
| 2 | 계약상대자 본사 상위 담당자와 접촉 |
| 3 | 계약관리자, 계약상대자, 구매자 등 회의 소집 (법률 대리인 포함 가능) |
| 4 | 계약 조항과 법률에 따른 조치 시행 (지체상금, 시정조치 통보) |
| 5 | 법적으로 계약종료(해지)를 위한 조치 시작 |

### 계약 해지 2가지 유형

| 유형 | 내용 |
|------|------|
| **편의적 해지** | 수요기관에 법률상 규정된 사유 발생 시 계약 종료 (정부정책 변경, 자금 중단 등) |
| **의무 불이행 해지** | 정상적 계약 이행·종결이 어렵다고 판단되는 경우 (매우 엄격한 절차) |

### 입찰참가자격 제한과 제재 조치

| 유형 | 내용 |
|------|------|
| Ineligibility | 특정 사유에 따라 자동적으로 자격 박탈 |
| Debarment | 법령 위반 확인 후 소명 거쳐 일정 기간 입찰참가 금지 |
| Suspension | 일시적 계약 이행 중단 또는 입찰참가자격 제한 |
| Reimbursement(과징금) | 경제적 부담 부과로 제재 효과 달성 |

국내: **부정당업자 제재** (최대 2년), 과징금 제도

### 계약목적물 인수
- 계약상대자가 이행 완료 통보 → 수요기관이 일정 기간(통상 **15일 내외**) 내 검사 완료
- 합격 시 납품확인서 발급 → 인도된 계약목적물이 발주기관 소유로 공식화

### 대금 지급
- 검사 합격 납품확인서 첨부하여 대금 지급 요청
- 지연 시 **지연 이자**(시중은행 평균 이자율 기준)까지 지급해야 함

## 핵심 암기 포인트

- 계약관리 6대 핵심 과업 숙지
- CMP 수립 원칙: **6하원칙**, **투입물→프로세스→결과물**
- 계약 모니터링: **7단계**
- 에스컬레이션 프레임워크: **5단계** (현장관리자 → 본사 → 회의 → 법적 조치 → 계약해지)
- 계약 해지 2유형: **편의적 해지 vs 의무 불이행 해지**
- 국내 부정당업자 제재: **최대 2년**
- 검사 완료 기간: 통상 **15일 내외**
- 대금 지연 시: **지연 이자 포함 지급**

## OX 퀴즈

**Q1.** 계약관리계획(CMP)은 모든 계약에 대해 수립해야 한다. → **X** (고위험 또는 매우 복잡한 조달사업에 한정)

**Q2.** 에스컬레이션 프레임워크의 주요 목표는 계약업체를 처벌하는 데 있다. → **X** (긍정적 관계를 유지하면서 정상 궤도로 되돌리는 것이 목적)

**Q3.** 편의적 계약 해지는 법률에 명확한 실행 근거를 명시해야 한다. → **O**

**Q4.** 부정당업자 제재는 최대 3년까지 가능하다. → **X** (최대 2년)

**Q5.** 대금 지급이 지연되어도 추가 비용 없이 원래 계약금액만 지급하면 된다. → **X** (지연 이자까지 포함하여 지급)

## 함정 노트

- ⚠️ 확정고정가격 계약에서 주요 내용 변경 시에도 **계약금액 조정이 사실상 불가능**할 수 있음
- ⚠️ 에스컬레이션의 목적: 처벌이 아닌 **정상 궤도 회복** (긍정적 관계 유지가 핵심)
- ⚠️ 계약분쟁 해결: 법적 소송보다 **상호 합의를 통한 해결이 권고** (비용, 시간, 관계 측면)
