# 제4절 입찰공고의 경제적 적정성 분석

## 4.1 경제성 분석 개요

입찰공고에서 제시된 사업예산과 요구사항을 고려하여 경제적으로 적정한 공급이 가능한지 분석하는 절차이다.

### 경제적 적정성 분석 8단계

| 단계 | 구분 | 내용 |
|:----:|------|------|
| 1 | 기초검토 | 사업예산 내역 확인, 부가가치세 포함 여부 |
| 2 | 가격분석 | 요구사항에서 유발되는 비용과 가격 합산 |
| 3 | 비용구성요소 분석 | 예정가격작성기준 원가계산기준에 따른 분석 |
| 4 | 계약조건 검토 | 준수요건 충족에 소요되는 비용 확인 |
| 5 | 납품일정 검토 | 납품기간 고려 시 추가 비용 확인 |
| 6 | 요구사항 충족수준 확인 | 기본적 충족 요건 검토 |
| 7 | 비교분석 | 유사 타입찰 계약목적물 가격과 비교 |
| 8 | 최종결정 | 숨은비용까지 고려한 종합적 비용적정성 검토 |

## 4.2 원가계산 방법

### 물품 원가계산(제조원가)

**예정가격 산정 기준**: 기획재정부 「예정가격작성기준」

| 비목 | 세부 구분 | 내용 | 산정 방법 |
|------|----------|------|-----------|
| **재료비** | 직접재료비 | 주요재료비 + 부분품비 | 재료량 x 단위당 가격 |
| | 간접재료비 | 소모재료비 + 소모공구비 + 포장재료비 | |
| **노무비** | 직접노무비 | 기본급 + 제수당 + 상여금 + 퇴직급여충당금 | 노무량 x 단위당 가격 |
| | 간접노무비 | 보조작업 노무자, 현장감독자 급여 | |
| **경비** | - | 재료비·노무비 외 제조원가 | 소요량 x 단위당 가격 |

**제조원가 = 재료비 + 노무비 + 경비**

### 일반관리비

- 기업 유지를 위한 관리활동 비용 (판매비 제외)
- 임원 급여, 사무실 직원 급여, 복리후생비, 여비, 감가상각비 등
- **제조업종별로 편차가 큼** → 해당 업종 일반관리비율 적용이 중요
- 기업 손익계산서 기준 산정

### 이윤

- 영업이익
- **(노무비 + 경비 + 일반관리비)의 25% 이하** (기술료·외주가공비 제외)

### 상여금·퇴직급여충당금 한도

| 항목 | 한도 |
|------|------|
| 상여금 | 기본급 기준 연 **400%** |
| 제수당, 퇴직급여충당금 | 근로기준법상 인정 범위 |

### 제조원가 계산 필요 서류

1. 규격서
2. 제작도면과 시방서
3. 공정도와 배치인원
4. 최근 **3개월분** 임금대장(월별집계표)
5. 기타 참고자료

### 경제성 분석 판단 기준

- 법정 계상비율 적용 일반관리비·이윤 포함 금액 > 사업예산 → **적정성 낮음**
- 사업예산 > 순원가(재료비+노무비+경비) → 직접 원가는 충당 가능 → 일반관리비·이윤 조정으로 대안 검토

## 핵심 암기 포인트

- 경제적 적정성 분석: **8단계**
- 제조원가 = **재료비 + 노무비 + 경비**
- 이윤 한도: **(노무비 + 경비 + 일반관리비)의 25% 이하**
- 상여금 한도: 기본급 기준 연 **400%**
- 일반관리비: **제조업종별로 비율이 다름** (편차가 큼)
- 제조원가 계산 필요 서류: **5가지** (규격서, 도면, 공정도, 임금대장, 참고자료)
- 임금대장: 최근 **3개월분**

## OX 퀴즈

**Q1.** 물품제조 원가계산에서 이윤은 제조원가 전체의 25%를 초과할 수 없다. → **X** (노무비+경비+일반관리비의 25% 이하, 기술료·외주가공비 제외)

**Q2.** 일반관리비율은 모든 제조업종에 동일하게 적용된다. → **X** (업종별로 편차가 크므로 해당 업종 비율 적용)

**Q3.** 상여금은 기본급의 연 500%까지 계상할 수 있다. → **X** (연 400%까지)

**Q4.** 사업예산이 순원가를 초과하면 해당 입찰의 경제적 적정성은 확보된 것이다. → **X** (일반관리비와 이윤을 포함한 종합적 분석이 필요)

**Q5.** 제조원가 계산을 위해 최근 6개월분 임금대장이 필요하다. → **X** (최근 3개월분)

## 함정 노트

- ⚠️ 이윤 계산 기준: **(노무비+경비+일반관리비)의 25%** → 재료비는 **포함되지 않음**, 기술료·외주가공비도 **제외**
- ⚠️ 물품제조 vs 물품구매의 예정가격 산정 차이: 제조 = **원가계산**, 구매 = **거래실례가격**
- ⚠️ 사업예산 vs 기초금액 vs 예정가격: 사업예산은 수요기관 배정액, 기초금액은 예정가격 산정 기준, 예정가격은 최종 상한금액
