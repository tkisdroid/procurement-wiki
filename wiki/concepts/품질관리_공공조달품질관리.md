# 제1절 공공조달 품질관리

## 1.1 품질관리 개요

- 1961년부터 규격 제정·납품검사 중심
- 2007년 **품질관리단** 신설
- 2014년 **조달품질원**으로 확대 (예방적 품질보증·품질경영 수준)

## 1.2 단계별 품질관리 활동

| 단계 | 내용 | 특징 |
|------|------|------|
| 진입단계 | 직접생산확인기준, 입찰참가자격등록 | 최소 품질·이행 여건 확인 |
| 계약단계 | 표준규격, 우수제품 구매규격, 제3자 인증, MAS 적격성평가 | 품질 변동 큰 물품 점검 |
| 이행(운영) | 조달물품 품질점검, 품질보증조달물품 지정 | 품질요건 충족 확인 |
| 완료(납품) | 조달청·전문기관·수요기관 검사 | 최종 품질 적정성 점검 |
| 사후관리 | 직접생산확인 점검, 품질신문고, 계약이행능력평가 | 하자 처리, 실태조사 |

### 최저가 낙찰제의 문제
- 저가경쟁 → 품질 저하 → 부실 납품 → 장기 비용 증가
- 대안: 적격심사, 종합심사, 품질·가격 균형 낙찰제 확대

### 품질신문고
- 수요자 원클릭 신고 → 조달청 해결 → 업체 피드백
