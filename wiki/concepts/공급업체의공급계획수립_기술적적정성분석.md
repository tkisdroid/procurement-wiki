# 제3절 입찰공고 기술적 적정성(내부 공급 역량) 분석

## 개요

입찰공고 문서의 기술적 요건을 분석하여, 공급업체가 기술입찰평가(TBE: Technical Bids Evaluation)를 통과할 수 있는지 **사전 충족 여부를 분석**하는 과정이다.

## 기술적 적정성 분석 9대 중점 사항

| 순서 | 분석 내용 |
|:----:|----------|
| 1 | **기술적 요건 준수 여부**: 기술사양, 규격 충족 + 납기 일정 이행 가능성 |
| 2 | **전문인력의 기술 역량**: 투입인력·수행조직의 자격, 인증, 수행 경험 |
| 3 | **종합적 타당성·위험성 평가**: 기간·예산 범위 내 이행 가능성, 외부 전문가 자문 |
| 4 | **증빙 정보·자료의 신뢰성**: 공공기관 발급 증빙, 제3자 인증서 유효기간 |
| 5 | **증빙문서의 명확성·연계성**: 시험성적서 항목·기준값의 동일 시점 측정 여부 |
| 6 | **평가항목별 비교우위**: 배점 높은 핵심 항목의 충족도 분석 |
| 7 | **유사 입찰 비교 분석**: 타 수요기관 입찰 사례의 기술적 요건과 비교 |
| 8 | **원부자재·부품 규격 부합 여부**: 입찰 담당자에게 문의·확인 |
| 9 | **종합 결과보고서 작성**: 항목별 충족 여부·수준을 정리하여 기록 관리 |

## 주요 확인 포인트

### 조달 방법 확인

- 다수공급자계약(MAS): 종합쇼핑몰 등록 필요 → 일반 입찰로 공급 불가
- 2,000만 원 이하: 1인 견적수의계약으로 전자조달시스템 미활용 가능
- 제조 공고: **제조업체만** 참여 가능 / 구매 공고: 제조업체 + 공급업체 모두 가능

### 기술적 미충족 시 결과

- 입찰참가자격 문제 발생
- 낙찰자 선정 후 계약 이행 과정에서 기능·성능 미달, 품질 저하
- 계약상 불이익, 부정당업자 제재 처분 가능

## 핵심 암기 포인트

- 기술적 적정성 분석: **9대 중점 사항**
- 시험성적서 확인: **동일 시점에서 측정**된 결과인지 검증 필요
- 자격·인증 유효기간: 입찰 마감일 ~ **계약 완료일까지** 유효해야 함
- 핵심 평가항목의 배점이 높은 항목 충족도가 낮으면: 기술적 경쟁 **열위**로 낙찰 어려움

## OX 퀴즈

**Q1.** 다수공급자계약(MAS) 물품은 일반 입찰공고를 통해서도 공급할 수 있다. → **X** (MAS 체결 후 종합쇼핑몰 등록 필요)

**Q2.** 기술적 요건 충족을 위한 자격·인증은 입찰 마감일까지만 유효하면 된다. → **X** (계약 완료일까지 유효해야 함)

**Q3.** 물품제조 입찰에는 제조업체와 공급업체 모두 참가할 수 있다. → **X** (제조업체만 참여 가능)

**Q4.** 시험성적서의 시험항목과 기준값은 동일 시점에서 측정된 결과여야 한다. → **O**

**Q5.** 기술적 요건 충족이 어려운 경우에도 가격 경쟁력이 있으면 낙찰 가능하다. → **X** (기술적 요건 미충족 시 입찰참가자격 문제 또는 평가 과정에서 배제)

## 함정 노트

- ⚠️ 물품제조 vs 물품구매 공고: 동일 세부 품명이라도 **제조 공고 시 제조업체만 참가 가능**
- ⚠️ 시험성적서 정합성: 이전 모델 기준 시험성적서 + 현행 모델 기준 인증서를 **혼합 제출하면 신뢰성 저하**
- ⚠️ 기술적 분석 결과 → 입찰참가 **여부 판단 근거**: 단순 참가 자격뿐 아니라 **계약 이행·납품 안정성**까지 고려
