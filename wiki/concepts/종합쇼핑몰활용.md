## 1. 다수공급자계약(MAS) 개요

「조달사업에 관한 법률」 제13조에 따라 수요기관이 선택할 수 있도록 **2인 이상**을 계약상대자로 하여 조달청장이 체결하는 **제3자를 위한 단가계약**

### 특징
- 반복적 구매수요가 있는 상용화·표준화 수준 높은 물품 대상
- 별도 입찰 절차 없이 수요기관 납품요구 시 **신속 납품**
- 조달청이 사전에 조달업체와 계약 체결

### MAS 입찰공고
- 공고기간: 기본 **10년** (신기술 물품 등은 5년)
- 차기 공고: 종료일 기준 **6개월 전**

### MAS 계약기간
- 기본: **3년**
- 연장 요건: 최근 3년 이내 해당 품목 **납품 실적** 필요
- 연장 신청: 계약 종료일 **2개월 전**부터 가능 (만료 후 불가)

## 2. MAS 절차 (6개 부문)

```
공고신청 → 적격성평가(또는 사전심사) → 가격협상 → 계약 → 종합쇼핑몰 등록
→ 수요기관 납품요구 → 납품 → 검사(검수) → 대금지급
```

### 적격성평가
- MAS 체결 업체의 적정 계약자격 보유 여부 평가
- 자기심사표 작성, 협상품목 등록 포함
- 경영상태: 신용평가등급 **B- 이상** (창업 3년 이내 기업은 면제)
- 세부품명별 납품실적 **3건 이상** 품목 1개 이상 보유

### 사전심사 (14개 특정 품명)
체육시설탄성포장재, 어린이놀이시설탄성포장재, 미끄럼방지포장재, 도막형바닥재, 철제가드레일, 철제도로중앙분리대, 철제교량난간, 알루미늄제도로중앙분리대, 알루미늄제교량난간, 낙석방지책, 차선분리대, 흙콘크리트, 구조물용충격완충장치, 단부처리용충격완충장치

### 가격협상
- 적격성평가 통과 후 MAS기초가격 작성 → MAS협상가격 작성
- 협상품목과 가격 확정 후 계약 체결

### 변경계약
- 품목추가, 단가변경 등으로 변경계약 소요 **상대적으로 빈번**
- 절차: 변경요청서 → 승인 → 변경계약서 초안 → 응답서 → 통보서

## 3. 우수제품 지정 신청

### 개요
- 기술력이 뛰어난 중소기업의 기술개발 촉진 목적
- **기술개발제품 우선구매 대상** 물품으로 지정
- 연간 **4회 내외** 지정계획 공고

### 지정 절차

```
지정계획 공고 확인 → 신규 신청서 작성·제출
→ 종합쇼핑몰지원센터 검토 (보완요청 가능)
→ 1차 심사 (기술수준 평가)
→ 특허적용여부확인서 제출
→ 현장 실태조사 + 규격서 검토
→ 2차 심사 → 우수제품 지정 + 지정증서 수여
```

### 지정 시작일 선택
- 지정일로부터 **30~180일** 지연 설정 가능
- 계약 준비 기간 확보, 불이익 방지 목적

## 핵심 암기 포인트

- MAS: 조달청이 체결하는 **제3자를 위한 단가계약**
- MAS 입찰공고 기간: **10년**, 계약기간: **3년**
- 계약기간 연장: 종료일 **2개월 전**부터, 최근 3년 내 납품실적 필요
- 적격성평가: 경영상태 신용등급 **B- 이상**, 납품실적 **3건+**
- 사전심사 대상: **14개** 특정 품명
- 우수제품: 연간 **4회 내외** 공고, 1차 심사(기술) → 2차 심사(현장)

## OX 퀴즈

**Q1.** 다수공급자계약의 입찰공고 기간은 기본 5년이다.
→ **X** (기본 10년이며, 신기술 물품 등은 5년이다)

**Q2.** MAS 계약기간 연장은 계약 만료 후에도 신청할 수 있다.
→ **X** (계약기간 만료 후에는 연장 신청이 불가능하며, 종료일 2개월 전부터 신청 가능하다)

**Q3.** 적격성평가에서 창업 3년 이내 기업은 신용평가등급 평가가 면제된다.
→ **O** (창업 3년 이내 기업은 경영상태 중 신용평가등급 평가가 면제된다)

**Q4.** 우수제품 지정 신청은 연간 1회만 가능하다.
→ **X** (연간 4회 내외의 지정계획 공고 일정에 맞추어 신청할 수 있다)

**Q5.** 사전심사 대상 물품은 적격성평가를 별도로 받아야 한다.
→ **X** (사전심사 대상 물품은 사전심사로 적격성평가에 갈음한다)

**Q6.** MAS 계약의 해지·해제는 조달업체만 신청할 수 있다.
→ **X** (조달청이 계약조건·납품조건 위반 시 직권으로 해지·해제할 수 있다)

## 함정 노트

⚠️ 협상품목 등록 시 목록정보시스템에 품목(물품식별번호 8자리)이 **미등록이면 목록화 절차를 먼저** 진행해야 한다

⚠️ MAS 계약 해지 신청 시 **미종결 납품요구 건**이 없는지 사전 확인 필요

⚠️ 우수제품 1차 심사 통과 후 **특허적용여부확인서**를 2차 심사 전에 반드시 제출해야 함 (필수 서류)
